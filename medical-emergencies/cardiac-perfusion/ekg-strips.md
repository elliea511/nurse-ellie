>
> - Count the rate
> - Decide if the rhythm is regular or irregular
> - Identify P waves, PR interval, and QRS width
> - Name the rhythm
> - Know the first nursing action
> - Treat the patient, not just the monitor

# Before You Name the Rhythm

| Step               | What to Check                                       | How to Check It                                              | Why It Matters                                               |
| ------------------ | --------------------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **1. Rate**        | How fast is the heart beating?                      | Count R waves on a 6-second strip × 10                       | Tells you if the rhythm is normal, too fast, or too slow     |
| **2. Rhythm**      | Regular or irregular?                               | Ask: “Can I walk it out?” Check if R waves are evenly spaced | Helps separate sinus rhythms from rhythms like A-fib         |
| **3. P Waves**     | Is the SA node starting the beat?                   | Look for a P wave before every QRS                           | No P waves or abnormal P waves can mean atrial, junctional, or ventricular rhythm |
| **4. PR Interval** | Is the impulse moving through the AV node normally? | Measure beginning of P wave to beginning of QRS              | Normal is **0.12-0.20 sec**; longer can mean AV block        |
| **5. QRS Width**   | Are the ventricles conducting normally?             | Measure the QRS complex                                      | Normal is **0.06-0.10 sec**; wide QRS often means ventricular problem |

![nm-how-to-read-EKG_line](https://www.nm.org/-/media/northwestern/healthbeat/images/healthy-tips/nm-how-to-read-ekg_line.jpg)

# EKG Parts to Know

| EKG Part        | What It Represents                            | Normal                   | Abnormal Clue                                                |
| --------------- | --------------------------------------------- | ------------------------ | ------------------------------------------------------------ |
| **P Wave**      | Atrial depolarization                         | Upright, before each QRS | Missing, hidden, inverted, or abnormal P waves can indicate dysrhythmia |
| **PR Interval** | Time from atria through AV node to ventricles | **0.12-0.20 sec**        | PR >0.20 sec = delayed conduction/first-degree AV block      |
| **QRS Complex** | Ventricular depolarization                    | **0.06-0.10 sec**        | Wide QRS = ventricular rhythm, PVC, VT, or conduction delay  |
| **ST Segment**  | Area to check for ischemia/injury             | Should stay on baseline  | ST elevation = myocardial injury/STEMI; ST depression = ischemia |
| **T Wave**      | Ventricular repolarization                    | Upright, not too tall    | Inverted T wave can indicate ischemia                        |

# Normal Sinus Rhythm

![EKG Normal Sinus Rythms, Sinus Bradycardia, & More – LevelUpRN](https://cdn.shopify.com/s/files/1/2630/9452/files/sinus-rhythms1-normal.jpg?v=1616525205)

### How to Recognize It

| Finding     | Expected in NSR     |
| ----------- | ------------------- |
| Rate        | **60-100 bpm**      |
| Rhythm      | Regular             |
| P waves     | Present and upright |
| PR interval | Normal              |
| QRS         | Normal/narrow       |

### What It Means

Normal sinus rhythm means the heart’s electrical impulse starts in the SA node, travels through the atria, pauses briefly at the AV node, then moves through the ventricles.

### Nursing Intervention

The patient is usually stable.

- Continue monitoring
- Assess pulse
- Document rhythm if needed

# Sinus Tachycardia

![EKG Normal Sinus Rythms, Sinus Bradycardia, & More – LevelUpRN](https://cdn.shopify.com/s/files/1/2630/9452/files/sinus-rhythms3-tachycardia.jpg?v=1616525205)

### How to Recognize It

| Finding     | Sinus Tachycardia        |
| ----------- | ------------------------ |
| Rate        | **Greater than 100 bpm** |
| Rhythm      | Regular                  |
| P waves     | Normal                   |
| PR interval | Normal                   |
| QRS         | Normal                   |

### What It Means

The heart is beating fast, but the electrical pathway is still normal. This is usually a response to something else going on in the body.

### Common Causes

| Cause                   | Why HR Increases                              |
| ----------------------- | --------------------------------------------- |
| Pain                    | Stress response increases heart rate          |
| Fever                   | Body needs more oxygen and circulation        |
| Anxiety                 | Sympathetic nervous system speeds up HR       |
| Dehydration/hypovolemia | Heart pumps faster to maintain blood pressure |
| Stimulants              | Caffeine, energy drinks, medications, etc.    |

### Nursing Intervention

Treat the **cause**, not just the number.

| Cause         | Nursing Action                                     |
| ------------- | -------------------------------------------------- |
| Pain          | Give prescribed pain medication                    |
| Fever         | Antipyretic, fluids, monitor temperature           |
| Dehydration   | Fluids if ordered, monitor BP and urine output     |
| Anxiety       | Calm environment, reassurance, breathing           |
| Cardiac cause | Beta blocker or calcium channel blocker if ordered |

# Sinus Bradycardia

![EKG Normal Sinus Rythms, Sinus Bradycardia, & More – LevelUpRN](https://cdn.shopify.com/s/files/1/2630/9452/files/sinus-rhythms2-bradycardia.jpg?v=1616525205)

### How to Recognize It

| Finding     | Sinus Bradycardia    |
| ----------- | -------------------- |
| Rate        | **Less than 60 bpm** |
| Rhythm      | Regular              |
| P waves     | Normal               |
| PR interval | Normal               |
| QRS         | Normal               |

### What It Means

The heart is beating slowly, but the electrical pathway is still normal.

This may be normal in:

- Athletes
- Sleeping patients
- Some patients taking beta blockers

### NCLEX Question to Ask

**Is the patient symptomatic?**

| Symptom        | Why It Matters                       |
| -------------- | ------------------------------------ |
| Hypotension    | Heart may be too slow to maintain BP |
| Dizziness      | Decreased blood flow to brain        |
| Syncope        | Severe decreased perfusion           |
| Pale/cool skin | Poor circulation                     |

### Nursing Intervention

| Patient Status  | Nursing Action                               |
| --------------- | -------------------------------------------- |
| Asymptomatic    | Monitor and assess pulse                     |
| Symptomatic     | Give **atropine** as ordered                 |
| Severe/unstable | Prepare for pacing if atropine does not work |

# Sinus Arrhythmia

![EKG Normal Sinus Rythms, Sinus Bradycardia, & More – LevelUpRN](https://cdn.shopify.com/s/files/1/2630/9452/files/sinus-rhythms4-arrythmia.jpg?v=1616525205)

### How to Recognize It

| Finding     | Sinus Arrhythmia |
| ----------- | ---------------- |
| Rate        | Variable         |
| Rhythm      | Irregular        |
| P waves     | Normal           |
| PR interval | Normal           |
| QRS         | Normal           |

### What It Means

The rhythm changes slightly with breathing. This can be normal, especially in younger or healthy patients.

### Nursing Intervention

- Continue monitoring
- Assess patient and pulse

# Atrial Flutter

<img src="https://cdn.shopify.com/s/files/1/2630/9452/files/atrial_rhythms-1.jpg" alt="EKG Atrial Dysrhythmia - Atrial Rythms – LevelUpRN" style="zoom:150%;" />

### How to Recognize It

| Finding | Atrial Flutter                                   |
| ------- | ------------------------------------------------ |
| Rate    | Atrial rate usually **250-300**                  |
| Rhythm  | Often regular, but ventricular response can vary |
| P waves | No true P waves                                  |
| Pattern | **Sawtooth/flutter waves**                       |
| QRS     | Usually normal                                   |

### What It Means

The atria are beating very fast. The AV node blocks some impulses, so not every atrial impulse reaches the ventricles.

### Biggest Risk

Atria do not squeeze normally
↓
Blood can pool
↓
Clots can form
↓
Clot can travel to brain
↓
**Stroke**

### Nursing Intervention

| Intervention             | Why                                  |
| ------------------------ | ------------------------------------ |
| Anticoagulants           | Help prevent clots and stroke        |
| Rate control             | Slows ventricular response           |
| Calcium channel blockers | Commonly used to slow rate           |
| Possible cardioversion   | May restore normal rhythm if ordered |

------

# Atrial Fibrillation

![Atrial Fibrillation • LITFL • ECG Library Diagnosis](https://litfl.com/wp-content/uploads/2018/08/Rhythm-Strip-LITFL-Atrial-Fibrillation.jpg)

### How to Recognize It

| Finding  | Atrial Fibrillation       |
| -------- | ------------------------- |
| Rhythm   | **Irregularly irregular** |
| P waves  | No true P waves           |
| Baseline | Scribbly/fibrillatory     |
| QRS      | Usually normal            |

### What It Means

The atria are quivering instead of contracting. Because they do not empty well, blood can pool and form clots.

### Biggest Risk

**Stroke**

Atria quiver
↓
Blood pools
↓
Clot forms
↓
Clot travels to brain
↓
Stroke

### Nursing Intervention

| Intervention           | Why                                           |
| ---------------------- | --------------------------------------------- |
| Anticoagulants         | Prevent clot formation and reduce stroke risk |
| Rate control           | Gives ventricles more time to fill            |
| Possible cardioversion | May restore normal rhythm if appropriate      |

### Key Point

A-fib = **irregular rhythm + no true P waves + stroke risk**

![Atrial Fibrillation vs. Flutter: Cardiocare Explains the Difference](https://cardiocarellc.com/app/uploads/2020/04/atrial-flutter-vs-afib-a-quick-comparison.webp)

# PAC: Premature Atrial Contraction

![EKG Atrial Dysrhythmia - Atrial Rythms – LevelUpRN](https://cdn.shopify.com/s/files/1/2630/9452/files/atrial_rhythms-2.jpg)

### How to Recognize It

| Finding     | PAC                                       |
| ----------- | ----------------------------------------- |
| Beat timing | Early beat                                |
| P wave      | May look different or be hidden in T wave |
| QRS         | Usually normal                            |

### What It Means

An early beat starts in the atria instead of waiting for the SA node.

### Nursing Intervention

Usually not dangerous by itself.

- Monitor rhythm
- Assess for triggers like caffeine, stress, or hypoxia

------

# PVC: Premature Ventricular Contraction

![EKG Interpretation: Ventricular Rhythms/Dysrhythmias Explained](https://cdn.shopify.com/s/files/1/2630/9452/files/ventricular-rhythm-1-pvc.jpg?v=1615827069)

### How to Recognize It

| Finding     | PVC                                     |
| ----------- | --------------------------------------- |
| Beat timing | Early beat                              |
| P wave      | Usually absent before PVC               |
| QRS         | Wide, bizarre, “ugly”                   |
| Rhythm      | Irregular because PVC interrupts rhythm |

### What It Means

An early beat starts in the ventricles.

### Why It Matters

Occasional PVCs may be tolerated, but frequent PVCs can become dangerous.

Watch for:

- Frequent PVCs
- Runs of PVCs
- PVCs with chest pain
- PVCs after MI
- PVCs progressing to VT or VF

### Nursing Intervention

- Assess patient
- Check oxygenation
- Monitor electrolytes, especially potassium and magnesium
- Report frequent or worsening PVCs

------

# Ventricular Tachycardia

![Ventricular tachycardia on EKG](https://cdn.shopify.com/s/files/1/2630/9452/files/ventricular-rhythm-2-vtach.jpg?v=1615827069)

### How to Recognize It

| Finding    | VT                       |
| ---------- | ------------------------ |
| Rate       | Usually greater than 100 |
| Rhythm     | Usually regular          |
| P waves    | Absent                   |
| QRS        | Wide                     |
| Appearance | Fast, wide rhythm        |

### What It Means

The ventricles are firing rapidly. This can severely decrease cardiac output because the ventricles do not have enough time to fill.

### Priority

## Check for a pulse first.

Do not treat only the monitor. Treat the patient.

### If Pulseless VT

This is cardiac arrest.

- Call for help
- Start CPR
- Defibrillate
- Follow ACLS

### If VT With a Pulse

Patient still has cardiac output.

- Assess stability
- Oxygen if needed
- Antidysrhythmics such as amiodarone or procainamide if ordered
- Possible synchronized cardioversion if unstable

------

# Ventricular Fibrillation

<img src="https://cdn.shopify.com/s/files/1/2630/9452/files/ventricular-rhythm-4-vfib.jpg?v=1615827068" alt="Ventricular fibrillation on EKG" style="zoom:200%;" />

### How to Recognize It

| Finding        | VF                |
| -------------- | ----------------- |
| Rhythm         | Chaotic           |
| P waves        | None              |
| QRS            | None identifiable |
| Cardiac output | None              |

### What It Means

The ventricles are quivering. They are not pumping blood.

### Nursing Priority

First make sure it is real:

- Check the patient
- Check responsiveness
- Check pulse
- Check leads quickly if patient is awake/talking

If true VF:

- Start CPR
- Defibrillate
- Follow ACLS

# Asystole

<img src="https://www.registerednursern.com/wp-content/uploads/2022/10/asystole.png" alt="Asystole ECG/EKG Rhythm Nursing Review" style="zoom: 150%;" />

### How to Recognize It

| Finding | Asystole  |
| ------- | --------- |
| Rhythm  | Flat line |
| P waves | None      |
| QRS     | None      |
| Pulse   | None      |

### What It Means

There is no ventricular electrical activity.

### NCLEX

**Do not shock asystole.**

### Nursing Intervention

- Check patient
- Start CPR
- Give epinephrine as ordered
- Look for reversible causes

# SVT: Supraventricular Tachycardia

<img src="https://www.ekgstripsearch.com/svt1.jpg" alt="EkG STRIP SEARCH; Supraventricular tachycardia SVT" style="zoom: 200%;" />

### How to Recognize It

| Finding | SVT                   |
| ------- | --------------------- |
| Rate    | 100-300               |
| Rhythm  | Regular               |
| P waves | Hidden or abnormal    |
| QRS     | Usually normal/narrow |

### What It Means

The rhythm starts above the ventricles and fires very fast. Because the rate is so fast, the heart may not have enough time to fill between beats.

### Stable SVT

Patient is still perfusing.

Examples:

- Alert
- BP okay
- No severe chest pain
- No severe shortness of breath

### Stable Treatment

1. **Vagal maneuvers**
2. **Adenosine**

### What Is a Vagal Maneuver?

A vagal maneuver stimulates the vagus nerve to slow conduction through the AV node.

Examples:

- Bearing down like having a bowel movement
- Forceful coughing
- Blowing through a straw

### Adenosine Note

Adenosine briefly blocks conduction through the AV node to reset the rhythm.

Important:

- Give rapid IV push
- Patient may feel flushed or uncomfortable
- Monitor may briefly look flat before rhythm returns

### Unstable SVT

Patient is not tolerating the rhythm.

Signs:

- Hypotension
- Chest pain
- Altered mental status
- Severe shortness of breath
- Shock signs

### Unstable Treatment

**Synchronized cardioversion**

# First-Degree AV Block

![1st degree AV block on EKG](https://cdn.shopify.com/s/files/1/2630/9452/files/av-blocks-1st-degree.jpg?v=1616011256)

### How to Recognize It

| Finding     | First-Degree AV Block    |
| ----------- | ------------------------ |
| Rhythm      | Regular                  |
| P waves     | Normal                   |
| PR interval | **Longer than 0.20 sec** |
| QRS         | Usually normal           |

### What It Means

The impulse is delayed at the AV node, but every impulse still gets through.

### Nursing Intervention

Usually monitor only.

------

# Second-Degree AV Block Type I: Wenckebach

![2nd degree AV block type 1 (Mobitz I, Wenckebach) on EKG](https://cdn.shopify.com/s/files/1/2630/9452/files/av-blocks-2nd-degree-type-1.jpg?v=1616011256)

### How to Recognize It

| Finding     | Wenckebach                  |
| ----------- | --------------------------- |
| PR interval | Gets longer, longer, longer |
| QRS         | Then one QRS drops          |
| Rhythm      | Irregular                   |

### Memory Trick

**Wenckebach = Watch it drop**

### What It Means

The AV node gets more delayed with each beat until one impulse does not make it through.

### Nursing Intervention

Usually monitor unless symptomatic.

------

# Second-Degree AV Block Type II

![2nd degree AV block type 2 (Mobitz II) on EKG](https://cdn.shopify.com/s/files/1/2630/9452/files/av-blocks-2nd-degree-type-2.jpg?v=1616011256)

### How to Recognize It

| Finding     | Mobitz II                            |
| ----------- | ------------------------------------ |
| PR interval | Stays the same when present          |
| QRS         | Random dropped QRS                   |
| Rhythm      | Irregular                            |
| Risk        | Can progress to complete heart block |

### What It Means

Some impulses suddenly fail to conduct to the ventricles. This is more dangerous than Type I.

### Nursing Intervention

- Monitor closely
- Prepare for pacing
- Notify provider
- May need pacemaker

------

# Third-Degree AV Block

<img src="https://cdn.shopify.com/s/files/1/2630/9452/files/av-blocks-3rd-degree.jpg?v=1616011255" alt="3rd degree AV block on EKG" style="zoom:150%;" />

### How to Recognize It

| Finding     | Third-Degree Block               |
| ----------- | -------------------------------- |
| P waves     | Present but not connected to QRS |
| QRS         | Present but independent          |
| PR interval | Random                           |
| Rhythm      | P waves and QRS do not match     |
| Rate        | Usually slow                     |

### What It Means

The atria and ventricles are not communicating. The atria have their own rhythm, and the ventricles have their own escape rhythm.

### Nursing Intervention

- Assess patient
- Prepare for transcutaneous pacing
- May need permanent pacemaker

# Pacemakers

A pacemaker is a device that sends an electrical impulse to make the heart beat when the heart’s own electrical system is too slow or unreliable.

Pacemakers are commonly used for:

- Symptomatic bradycardia
- Second-degree AV block Type II
- Third-degree heart block
- Sick sinus syndrome

------

## Pacemaker Spike

A pacemaker spike is a thin vertical line on the EKG.

It means the pacemaker fired.

What should happen next depends on the chamber being paced.

| Type of Pacing         | What You See                             |
| ---------------------- | ---------------------------------------- |
| **Atrial pacing**      | Pacemaker spike before a P wave          |
| **Ventricular pacing** | Pacemaker spike before a QRS             |
| **AV pacing**          | Spike before P wave and spike before QRS |

![EKGDX - The new era of EKG learning](https://ekgdx-prod.s3.amazonaws.com/article_pics/3598f87f-b74c-4785-baaf-03fabb6a4d25.png)

------

## Capture

Capture means the heart responded to the pacemaker impulse.

| If Pacemaker Fires       | Expected Response |
| ------------------------ | ----------------- |
| Atrial pacing spike      | P wave follows    |
| Ventricular pacing spike | QRS follows       |

### Easy Way to Remember

**Spike + beat after it = capture**



------

## Failure to Capture

### What You See

Pacemaker spike is present, but no P wave or QRS follows.

Example:

Pacemaker spike
↓
Nothing happens

<img src="https://ekgdx-prod.s3.amazonaws.com/uploads/Failure%20to%20Capture%20-%20EKGDX.png" alt="EKGDX - The new era of EKG learning" style="zoom:150%;" />

### What It Means

The pacemaker fired, but the heart did not respond.

### Why It Matters

The patient may become bradycardic, hypotensive, dizzy, or unstable because the pacemaker is not creating effective beats.

### Nursing Actions

- Assess the patient first
- Check pulse and blood pressure
- Notify provider
- Prepare for pacemaker troubleshooting
- May need temporary pacing or adjustment

### NCLEX Clue

**Pacemaker spike not followed by QRS = failure to capture**

------

## Failure to Pace

### What You See

The pacemaker should be firing, but there are no pacemaker spikes when expected.



### What It Means

The pacemaker is not sending impulses.

### Nursing Actions

- Assess patient
- Check pulse
- Check pacemaker settings/battery if applicable
- Notify provider

### NCLEX Clue

**No spike when one should be there = failure to pace**

------

## Failure to Sense

### What You See

Pacemaker spikes occur at the wrong time, even when the patient’s heart is already beating.

<img src="https://lh6.googleusercontent.com/-LjN8bHwYKTRahyml9xD0n0tFWrsI36YsGMOrGS-7A4iK49yZ5_FiEMt0U4bs7_6iwefiz-5xHiNPsVvojkQD5dnUMOwYxRYnhjtfcZrUsSI_75Wfb5loBumjG1MOfhRENpzubpl" alt="img" style="zoom:150%;" />

### What It Means

The pacemaker is not sensing the patient’s own rhythm correctly.

### Why It Matters

A spike can fall in a dangerous place, especially near the T wave, and may trigger a dysrhythmia.

### Nursing Actions

- Assess patient
- Monitor rhythm
- Notify provider
- Prepare for pacemaker adjustment

### NCLEX Clue

**Pacemaker spike shows up randomly or too close to the patient’s own beat = failure to sense**

------

## Pacemaker Problems Quick Chart

| Problem                | What You See           | Meaning                                      |
| ---------------------- | ---------------------- | -------------------------------------------- |
| **Failure to capture** | Spike but no QRS after | Pacemaker fired, heart did not respond       |
| **Failure to pace**    | No spike when needed   | Pacemaker did not fire                       |
| **Failure to sense**   | Spike at wrong time    | Pacemaker did not sense the heart’s own beat |

# High-Yield NCLEX Clues

| Finding                          | Think                    |
| -------------------------------- | ------------------------ |
| Sawtooth pattern                 | Atrial flutter           |
| Scribbly baseline                | Atrial fibrillation      |
| Irregularly irregular            | Atrial fibrillation      |
| Wide ugly early beat             | PVC                      |
| Fast wide rhythm                 | Ventricular tachycardia  |
| Chaotic rhythm                   | Ventricular fibrillation |
| Flat line                        | Asystole                 |
| PR >0.20 sec                     | First-degree AV block    |
| Longer, longer, longer, drop     | Wenckebach               |
| Same PR, random dropped QRS      | Mobitz II                |
| P waves and QRS do not match     | Third-degree heart block |
| Spike with no QRS after          | Failure to capture       |
| No pacemaker spike when expected | Failure to pace          |
| Spike at the wrong time          | Failure to sense         |

# Absolute Exam Must-Knows

- Count HR using **R waves × 10**
- Normal PR = **0.12-0.20 sec**
- Normal QRS = **0.06-0.10 sec**
- ST elevation = **myocardial injury/STEMI**
- ST depression or T-wave inversion = **ischemia**
- Wide QRS = **ventricular conduction problem**
- VT = **check pulse first**
- VF = **check patient, CPR, defibrillate**
- Asystole = **CPR + epinephrine, do not shock**
- SVT stable = **vagal maneuvers, adenosine**
- SVT unstable = **synchronized cardioversion**
- A-flutter = **sawtooth**
- A-fib = **irregularly irregular**
- Wenckebach = **longer, longer, drop**
- Mobitz II = **same PR, random drop**
- Third-degree block = **P waves and QRS do not communicate**
- Pacemaker spike with no QRS = **failure to capture**
