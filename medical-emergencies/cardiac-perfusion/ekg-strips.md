# EKG Strips Interpretation Guide

> **Exam Focus (per instructor):**
>
> * Identify the rhythm
> * Count the heart rate
> * Measure the PR interval & QRS
> * Know the FIRST nursing intervention
> * Know which medication or treatment is indicated

---

# Before You Name the Rhythm (ALWAYS use this order)

### 1. Rate

* **6-second strip:** Count **R waves × 10**
* Example:

  * 7 R waves = **70 bpm**
  * 10 R waves = **100 bpm**
  * 5 R waves = **50 bpm**

### 2. Rhythm

Ask:

**"Can I walk it out?"**

* Same distance between every R wave = **Regular**
* Different spacing = **Irregular**

### 3. P Waves

* Present?
* One P for every QRS?
* Upright?

### 4. PR Interval

Measure beginning of **P** to beginning of **QRS**

Normal:
**0.12-0.20 sec**
(3-5 small boxes)

### 5. QRS Width

Normal:
**0.06-0.10 sec**
(<3 small boxes)

Wide QRS = ventricular conduction problem

---

# Know the Parts of the EKG

### P Wave

* Atrial depolarization
* Atria contract

---

### PR Interval

Normal:
**0.12-0.20 sec**

Represents:

* Time impulse travels from atria → AV node → ventricles

Long PR (>0.20)
= First-degree AV block

---

### QRS Complex

Represents:

* Ventricular depolarization

Normal:
**0.06-0.10 sec**

Wide QRS
Think:

* Ventricular rhythm
* Bundle branch block
* PVC
* VT

---

### ST Segment

#### ST Elevation

= Myocardial injury (STEMI)

#### ST Depression

= Ischemia
May also occur with digoxin

---

### T Wave

Represents:

* Ventricular repolarization

Normal:

* Bigger than P wave
* Less than 5 small boxes tall

Inverted T wave
= Ischemia

# Rhythm Recognition

---

# Normal Sinus Rhythm (NSR)

### Recognition

✔ Rate 60-100

✔ Regular

✔ Normal P waves

✔ Normal PR

✔ Normal QRS

### Intervention

* Continue monitoring
* Assess pulse

---

# Sinus Tachycardia

### Recognition

Rate:
**>100**

Everything else normal.

Think:

* Pain
* Anxiety
* Fever
* Exercise
* Hypovolemia
* Stimulants (energy drinks, caffeine)

### NCLEX

Treat the **CAUSE**.

### Intervention

* Pain meds
* Antipyretics
* Fluids if dehydrated
* Beta blocker if ordered
* Calcium channel blocker if ordered

---

# Sinus Bradycardia

### Recognition

Rate:
**<60**

Everything else normal.

### NCLEX

If athlete and asymptomatic
→ Usually normal.

If symptomatic:

* Hypotension
* Dizziness
* Syncope
* Pale

### Intervention

Symptomatic:
→ **Atropine**

Otherwise:
Monitor

---

# Sinus Arrhythmia

### Recognition

Rate:
Variable

Rhythm:
Irregular

Everything else normal.

Usually occurs with breathing.

### Intervention

Monitor

Usually doesn't require treatment.

---

# Atrial Flutter

### Recognition

✔ Sawtooth pattern

Atrial rate:
250-300

PR:
Absent

### NCLEX Clue

Looks like:

**Butterfly wings**
or
**Saw blade**

### Biggest Risk

Blood pools

↓

Clots

↓

Stroke

### Intervention

* Anticoagulants
* Rate control (Calcium channel blockers)
* Possible cardioversion

---

# Atrial Fibrillation

### Recognition

No P waves

"Scribble" baseline

Irregularly irregular

### Biggest Risk

Stroke

### Intervention

* Anticoagulants
* Rate control
* Possible cardioversion

---

# Flutter vs Fib

Flutter

Saw teeth

Regular

Fib

Scribbles

Irregular

---

# PAC (Premature Atrial Contraction)

### Recognition

Early beat

P wave often hidden inside previous T wave

Pause afterward

### Intervention

Usually monitor.

---

# PVC (Premature Ventricular Contraction)

### Recognition

Wide ugly QRS

Early beat

No P wave before PVC

### NCLEX

One PVC?
Usually okay.

Several PVCs in a row?

Can progress to:

* Ventricular Tachycardia
* Ventricular Fibrillation

### Intervention

Monitor closely.

---

# Ventricular Tachycardia (VT)

### Recognition

Rate:

> 100

Wide QRS

No P waves

Regular

### PRIORITY

## CHECK FOR A PULSE FIRST!

Treat the patient—

NOT the monitor.

### Pulseless VT

* CPR
* Defibrillate
* ACLS

### Pulse Present

* Amiodarone
* Procainamide
* Possible synchronized cardioversion

---

# Ventricular Fibrillation (VF)

### Recognition

Chaotic

No QRS

No P waves

Patient has NO cardiac output.

### PRIORITY

Before assuming arrest:

✔ Check patient

✔ Check leads

If truly VF:

* CPR
* Defibrillation
* ACLS

---

# Asystole

### Recognition

Flat line

No rhythm

### NCLEX

**DO NOT SHOCK**

### Intervention

* CPR
* Epinephrine
* Find reversible causes (Hs & Ts)

---

# Supraventricular Tachycardia (SVT)

### Recognition

Rate:
100-300

Regular

P waves hidden

### Intervention

Stable

1. Vagal maneuvers
2. Adenosine

Unstable

→ Synchronized cardioversion

### Remember

Adenosine may briefly make the monitor appear flat before normal rhythm returns.

---

# AV Blocks

---

# First-Degree AV Block

### Recognition

Only finding:

PR interval

**>0.20 sec**

Everything else normal.

### Intervention

Monitor

---

# Second-Degree Type I (Wenckebach)

### Recognition

PR gets

Longer

Longer

Longer

DROP!

Mnemonic:

**"Watch it drop."**

### Intervention

Monitor

---

# Second-Degree Type II

### Recognition

PR stays the SAME

Suddenly drops QRS

No warning

### NCLEX

More dangerous.

Can become complete heart block.

### Intervention

Prepare for pacemaker.

---

# Third-Degree (Complete Heart Block)

### Recognition

P waves and QRS have

NO relationship.

PR interval random.

Slow ventricular rate.

### Intervention

* Transcutaneous pacing
* Permanent pacemaker

---

# Pacemakers

### Failure to Capture

Pacemaker spike

NO QRS afterward

NCLEX answer:
**Failure to Capture**

---

# Shockable vs Non-Shockable

## SHOCK

✔ Ventricular Fibrillation

✔ Pulseless Ventricular Tachycardia

---

## DO NOT SHOCK

✔ Asystole

---

# Medications to Know

| Rhythm                  | Medication                               |
| ----------------------- | ---------------------------------------- |
| Symptomatic Bradycardia | Atropine                                 |
| SVT                     | Adenosine                                |
| Atrial Flutter/Fib      | Anticoagulants, Calcium Channel Blockers |
| VT with Pulse           | Amiodarone or Procainamide               |
| Asystole                | Epinephrine                              |

---

# High-Yield NCLEX Clues

| Finding                  | Think               |
| ------------------------ | ------------------- |
| Saw teeth                | Atrial Flutter      |
| Scribbles                | Atrial Fibrillation |
| Wide ugly beat           | PVC                 |
| Wide fast rhythm         | VT                  |
| Chaotic rhythm           | VF                  |
| Flat line                | Asystole            |
| PR >0.20                 | First-degree block  |
| Longer...longer...DROP   | Wenckebach          |
| Same PR then drop        | Mobitz II           |
| P waves don't match QRS  | Third-degree block  |
| Early P buried in T wave | PAC                 |

---

# Absolute Exam Must-Knows

Count HR using **R waves × 10**

Normal PR = **0.12-0.20 sec**

Normal QRS = **0.06-0.10 sec**

ST elevation = **MI/STEMI**

ST depression/T-wave inversion = **Ischemia**

Wide QRS = **Ventricular rhythm**

VT → **Check pulse FIRST**

VF → **Assess patient, then CPR + Defibrillate**

Asystole → **CPR + Epinephrine (NO shock)**

SVT → **Vagal maneuvers → Adenosine**

Flutter = **Saw teeth**

Fib = **Scribbles**

Wenckebach = **Longer, Longer, DROP**

Mobitz II = **Same PR, then DROP**

Third-degree = **Complete heart block → Pacemaker**
