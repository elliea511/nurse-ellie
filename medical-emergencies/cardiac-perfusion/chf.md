## Acute CHF / Pulmonary Edema

Congestive heart failure means the heart cannot pump effectively enough to meet the body’s needs.

Fluid can back up into the lungs or the body.

When fluid backs up into the lungs, it can become an emergency called **pulmonary edema**.

---

# CHF Basics

| Concept | What It Means |
| --- | --- |
| CHF | Heart cannot pump effectively enough for body needs |
| Fluid backup | Fluid can back up into lungs or body tissues |
| Pulmonary edema | Fluid in the lungs causing impaired gas exchange |
| Emergency priority | Improve oxygenation |

<div class="tip-box">
<strong>Priority Tip:</strong> If the client has severe shortness of breath, crackles, low O₂ saturation, and pink frothy sputum, think <strong>acute pulmonary edema</strong>. This is an oxygenation emergency.
</div>

---

# Left-Sided vs Right-Sided Heart Failure

| Type | Memory Trick | Main Backup Location | Main Symptoms |
| --- | --- | --- | --- |
| Left-sided heart failure | **Left = Lungs** | Blood backs up into lungs | Dyspnea, crackles, cough, orthopnea, low O₂ saturation |
| Right-sided heart failure | **Right = Rest of body** | Blood backs up into body/venous system | JVD, peripheral edema, weight gain, ascites, hepatomegaly |

---

## Left-Sided Heart Failure

Left-sided heart failure causes lung symptoms.

### Why It Happens

The left ventricle cannot pump blood forward effectively.

| Backup Pathway |
| --- |
| Left ventricle |
| Left atrium |
| Pulmonary veins |
| Lungs |

### Left-Sided Signs and Symptoms

| Finding | Why It Happens / What It Means |
| --- | --- |
| Dyspnea | Fluid backs up into lungs |
| Crackles/rales | Fluid in alveoli/lung tissue |
| Cough | Lung congestion |
| Orthopnea | Trouble breathing when lying flat |
| Paroxysmal nocturnal dyspnea | Wakes up short of breath at night |
| Low oxygen saturation | Impaired gas exchange |
| Restlessness | Early sign of hypoxia |
| Fatigue | Decreased cardiac output |
| Pink frothy sputum | Severe pulmonary edema |

<div class="tip-box">
<strong>NCLEX Tip:</strong> Restlessness can be an early sign of hypoxia. Do not ignore new anxiety/restlessness in a client with CHF.
</div>

---

## Right-Sided Heart Failure

Right-sided heart failure causes venous congestion in the body.

### Right-Sided Signs and Symptoms

| Finding | Why It Happens / What It Means |
| --- | --- |
| Jugular vein distention | Venous congestion |
| Peripheral edema | Fluid backs up into tissues |
| Pitting edema | Fluid overload in extremities |
| Weight gain | Fluid retention |
| Ascites | Fluid in abdomen |
| Hepatomegaly | Liver congestion |
| Enlarged abdomen | Fluid backup/ascites |
| Nausea | GI/liver congestion |

---

# Acute Pulmonary Edema

## What It Is

Pulmonary edema occurs when fluid builds up in the lungs.

This is a medical emergency because gas exchange is impaired.

## Recognize Acute Pulmonary Edema

| Sign/Symptom | What To Think |
| --- | --- |
| Severe shortness of breath | Oxygenation problem |
| Crackles | Fluid in lungs |
| Frothy sputum | Pulmonary edema |
| Pink frothy sputum | Severe pulmonary edema |
| Anxiety/restlessness | Possible hypoxia |
| Tachycardia | Compensation for low oxygen/cardiac output |
| Low oxygen saturation | Impaired gas exchange |
| Sitting upright/tripod positioning | Trying to breathe easier |
| Cool, clammy skin | Poor perfusion/stress response |

## Priority Nursing Action

**Improve oxygenation.**

<div class="priority-box">
<strong>What do I do first?</strong><br>
Sit the client up in <strong>high Fowler’s</strong> and apply oxygen. Then continue assessment, notify provider/rapid response if severe, and give prescribed medications.
</div>

---

# Nursing Care for Acute CHF / Pulmonary Edema

| Priority | Nursing Action | Why |
| --- | --- | --- |
| 1 | Place in high Fowler’s | Helps lung expansion and decreases venous return |
| 2 | Apply oxygen | Improves oxygenation |
| 3 | Assess lung sounds | Monitors fluid in lungs |
| 4 | Monitor oxygen saturation | Tracks oxygenation status |
| 5 | Notify provider or rapid response if severe | Pulmonary edema can worsen quickly |
| 6 | Administer diuretics as prescribed | Removes excess fluid |
| 7 | Monitor vital signs | Detects instability |
| 8 | Monitor intake and output | Evaluates fluid balance |
| 9 | Monitor urine output | Shows response to diuretics and kidney perfusion |
| 10 | Monitor potassium and renal labs | Diuretics can affect electrolytes/kidneys |
| 11 | Reduce anxiety | Decreases oxygen demand |
| 12 | Prepare for additional respiratory support if needed | Client may need higher-level oxygen support |

---

# CHF Treatment: “Pump It and Pee It”

CHF treatment often focuses on helping the heart pump and removing extra fluid.

| Goal | Medication Example | Why |
| --- | --- | --- |
| Help the heart pump | Digoxin | Slows and strengthens heartbeat |
| Remove extra fluid | Furosemide/Lasix | Decreases fluid overload |

---

# Digoxin

## What It Does

Digoxin slows and strengthens the heartbeat.

It acts on the SA node and AV node.

| Digoxin Effect | Why It Matters |
| --- | --- |
| Slows heart rate | Helps control rapid rhythms |
| Strengthens contraction | Improves cardiac output |
| Acts on SA and AV nodes | Affects heart rate/conduction |

## Uses

| Used For |
| --- |
| Heart failure |
| Atrial fibrillation |
| Atrial flutter |
| Some supraventricular dysrhythmias |

---

## Before Giving Digoxin

Check the **apical pulse for 1 full minute**.

| Client | Hold Digoxin If |
| --- | --- |
| Adult | Pulse is below 60 |
| Infant | Pulse is below 100 |
| Toddler | Pulse is around below 90 |
| Any client | Signs of toxicity are present |
| Any client | Digoxin level is too high |

<div class="tip-box">
<strong>NCLEX Tip:</strong> If the question asks what to do before giving digoxin, choose <strong>check apical pulse for 1 full minute</strong>.
</div>

---

## Digoxin Levels

| Level | Meaning |
| --- | --- |
| 0.8–2.0 ng/mL | Therapeutic |
| Greater than 2.0 ng/mL | Hold and notify provider |
| Around 2.5 ng/mL | Toxicity concern |

## Antidote

| Medication | Use |
| --- | --- |
| Digibind | Antidote for serious digoxin toxicity |

---

## Digoxin Toxicity

| Toxicity Finding | NCLEX Cue |
| --- | --- |
| Nausea | GI symptom of toxicity |
| Vomiting | GI symptom of toxicity |
| Anorexia | Loss of appetite can be an early clue |
| Bradycardia | Digoxin slows the heart |
| Dysrhythmias | Dangerous toxicity finding |
| Green/yellow halos | Classic test clue |
| Vision changes | Classic test clue |
| Confusion | Neuro symptom |
| Dizziness | Safety concern |

<div class="tip-box">
<strong>NCLEX Clue:</strong> If the client says they see <strong>green or yellow halos</strong>, think <strong>digoxin toxicity</strong>.
</div>

---

# Potassium and Digoxin

Low potassium increases the risk of digoxin toxicity.

This is important because clients may take digoxin with furosemide/Lasix.

Lasix can lower potassium.

| Situation | Risk |
| --- | --- |
| Digoxin + normal potassium | Lower toxicity risk |
| Digoxin + low potassium | Higher toxicity risk |
| Lasix use | Can cause hypokalemia |
| Low potassium + digoxin | Increased toxicity risk |

<div class="priority-box">
<strong>Priority Tip:</strong> Check potassium before giving digoxin if the client is also taking Lasix or has symptoms of digoxin toxicity.
</div>

---

# Diuretics in CHF

## Diuretic Comparison

| Type | Examples | Main Concerns |
| --- | --- | --- |
| Thiazide diuretics | Hydrochlorothiazide, Diuril | Hypokalemia, orthostatic hypotension |
| Loop diuretics | Furosemide/Lasix | Hypokalemia, hyponatremia |
| Potassium-sparing diuretics | Spironolactone/Aldactone | Hyperkalemia |

---

## Spironolactone Action

Spironolactone blocks aldosterone.

Aldosterone tells the kidneys to hold sodium and water and get rid of potassium.

| If Aldosterone Is Blocked | Result |
| --- | --- |
| Sodium and water are excreted | Less fluid overload |
| Potassium is retained | Risk for hyperkalemia |

<div class="tip-box">
<strong>NCLEX Tip:</strong> Spironolactone is potassium-sparing. Avoid extra potassium or salt substitutes unless prescribed.
</div>

---

# Potassium Teaching

## High-Potassium Foods

Know these for clients on potassium-wasting diuretics.

| High-Potassium Foods |
| --- |
| Bananas |
| Oranges |
| Melons |
| Apricots |
| Prunes |
| Baked potatoes |
| Beans |
| Squash |
| Meats |

## Who Needs Potassium Teaching?

| Medication Situation | Teaching Focus |
| --- | --- |
| Taking Lasix or thiazide | May need potassium-rich foods if ordered |
| Taking spironolactone | Avoid too much potassium unless prescribed |
| Taking digoxin + Lasix | Monitor potassium because low K⁺ increases digoxin toxicity risk |

# CHF Diagnostics

| Test | What It Shows |
| --- | --- |
| BNP | Elevated in heart failure |
| Echocardiogram | Heart structure, valves, ejection fraction |
| MUGA scan | Ejection fraction and ventricular wall motion |
| Chest x-ray | Fluid in lungs, enlarged heart |
| CT/MRI | Heart structure |
| ECG | Dysrhythmias or cardiac strain |

---

# BNP

BNP is released when the ventricles are stretched from pressure and fluid overload.

## BNP Values

| BNP | Meaning |
| --- | --- |
| Less than 100 | Normal |
| 100–300 | Mild |
| 300–600 | Moderate |
| 600–900 | Severe |
| Greater than 900 | Very concerning |

<div class="tip-box">
<strong>NCLEX Tip:</strong> BNP helps identify heart failure/fluid overload. The higher the BNP, the more concerning the heart failure picture is.
</div>

# Most Important “What Do I Do First?” Points

| Situation | First / Priority Action |
| --- | --- |
| Acute pulmonary edema | Improve oxygenation |
| Severe SOB with crackles | High Fowler’s and oxygen |
| Before giving digoxin | Check apical pulse for 1 full minute |
| Digoxin level > 2.0 ng/mL | Hold and notify provider |
| Green/yellow halos | Hold digoxin and notify provider |
| Lasix given | Monitor urine output, potassium, BP, renal labs |
| Spironolactone ordered | Monitor for hyperkalemia |
| CHF worsening | Monitor lung sounds, O₂ sat, daily weight, I&O |
