# Increased ICP and Hydrocephalus

---

## Increased Intracranial Pressure

| Concept | Simple Explanation |
| ------- | ------------------ |
| Increased ICP | Too much pressure inside the skull |
| Skull | A **rigid box** that does not stretch like skin |
| Main danger | Pressure decreases blood flow and oxygen to the brain |
| Priority concern | Brain perfusion and preventing herniation |

## What Is Inside the Skull?

| Skull Content | Why It Matters |
| ------------- | -------------- |
| Brain tissue | Swelling takes up space |
| Blood | Bleeding or increased blood volume raises pressure |
| Cerebrospinal fluid, or CSF | Too much CSF can raise pressure |

If brain tissue, blood, or CSF increases, pressure inside the skull rises.

---

# Causes of Increased ICP

| Cause | How It Raises Pressure |
| ----- | ---------------------- |
| Brain swelling | Takes up extra space inside skull |
| Head injury | Can cause swelling or bleeding |
| Bleeding in the brain | Adds blood volume inside skull |
| Tumor | Takes up space and compresses tissue |
| Stroke | Can cause swelling or bleeding |
| Infection | Can cause inflammation and swelling |
| Hydrocephalus | Excess CSF builds up |
| Excess CSF | Fluid volume increases pressure |

---

# Why Increased ICP Is Dangerous

| Problem | Result |
| ------- | ------ |
| Increased pressure | Decreased blood flow to brain |
| Decreased blood flow | Less oxygen reaches brain tissue |
| Brain tissue damage | Neuro changes, seizures, or decreased LOC |
| Severe pressure | Herniation, respiratory changes, death |

<div class="tip-box">
<strong>Priority Tip:</strong> The brain cannot tolerate decreased oxygen and perfusion for long. With ICP questions, think: protect airway, support oxygenation, maintain cerebral perfusion, and report worsening neuro changes.
</div>

---

# Normal ICP

| ICP Value | Meaning | Nursing Concern |
| --------- | ------- | --------------- |
| 0–15 mm Hg | Normal ICP | Continue monitoring |
| 20 mm Hg or higher for more than 5 minutes | Life-threatening emergency | Report immediately / intervene quickly |

---

# Recognizing Increased ICP

## Earliest Sign

| Earliest Sign | What It May Look Like |
| ------------- | --------------------- |
| **Change in level of consciousness** | Restlessness, confusion, drowsiness, irritability, decreased responsiveness |

<div class="nclex-box">
<strong>NCLEX Tip:</strong> A change in LOC is usually the earliest and most important sign of increased ICP. Do not wait for late vital sign changes.
</div>

## Early vs Late Findings

| Earlier Findings | Later / More Serious Findings |
| ---------------- | ----------------------------- |
| Change in LOC | Cushing’s triad |
| Restlessness | Bradycardia |
| Confusion | Hypertension |
| Drowsiness | Widened pulse pressure |
| Irritability | Bradypnea |
| Headache | Cheyne-Stokes respirations |
| Nausea/vomiting | Fixed and dilated pupils |
| Pupillary changes | Abnormal posturing |
| Weakness | Severe headache |
| Slurred speech | Projectile vomiting |
| Seizures | Decreased responsiveness/coma |
| Papilledema | Herniation risk |

---

# Cushing’s Triad

Cushing’s triad is a **late sign** of increased ICP.

| Finding | What Happens | Why It Matters |
| ------- | ------------ | -------------- |
| Systolic hypertension | Systolic BP increases | Body tries to perfuse brain |
| Widened pulse pressure | Difference between systolic and diastolic BP widens | Worsening pressure pattern |
| Bradycardia | Pulse decreases | Late neuro deterioration sign |
| Respiratory depression | Respirations decrease or become abnormal | Brainstem involvement concern |

<div class="tip-box">
<strong>Priority Tip:</strong> Cushing’s triad is not an early warning. It means the client is already in serious danger.
</div>

---

# Increased ICP vs Shock

| Finding | Increased ICP | Shock |
| ------- | ------------- | ----- |
| Blood pressure | Systolic BP increases | BP decreases |
| Pulse | Decreases | Increases |
| Respirations | Decrease or become irregular | Increase |
| Priority concern | Brain pressure/perfusion | Tissue perfusion |
| NCLEX clue | Widened pulse pressure + bradycardia | Hypotension + tachycardia |

---

# Level of Consciousness Terms

| Term | Meaning |
| ---- | ------- |
| Full consciousness | Awake, alert, oriented, interacts with environment |
| Confused | Lacks ability to think clearly and rapidly |
| Disoriented | Does not recognize place or person |
| Lethargic | Awakens easily but has limited responsiveness |
| Obtunded | Sleeps and, once aroused, has limited interaction |
| Stupor | Requires stimulation to arouse |
| Coma | Vigorous stimulation produces no response |

---

# Glasgow Coma Scale Basics

You may not need to calculate the full score, but know what the ranges mean.

| GCS Score | Severity | NCLEX Meaning |
| --------- | -------- | ------------- |
| 13–15 | Mild head injury | Monitor closely |
| 9–12 | Moderate head injury | Increased concern |
| 3–8 | Severe head injury | Airway risk / possible intubation concern |

---

# Diagnostic Tests for Increased ICP

| Test Type | Examples | Why It May Be Ordered |
| --------- | -------- | --------------------- |
| Blood gases | ABGs | Evaluate oxygenation/ventilation |
| Blood count | CBC | Check infection/bleeding status |
| Clotting labs | Coagulation profile | Bleeding risk or anticoagulant concerns |
| Electrolytes | Sodium, potassium, etc. | Neuro changes can worsen with imbalances |
| Osmolality | Serum osmolality, urine osmolality | Fluid balance and mannitol/hypertonic therapy monitoring |
| Urine testing | Urinalysis | Hydration/kidney status |
| Imaging | CT scan of the head, MRI | Look for bleeding, swelling, tumor, stroke, hydrocephalus |
| Brain activity | EEG | Evaluate seizure activity |

---

# Priority Assessment for Increased ICP

| Assess | Why It Matters |
| ------ | -------------- |
| Level of consciousness | Earliest and most important change |
| Pupils | Can show worsening pressure/brain involvement |
| Motor strength | Detects weakness or new neuro deficit |
| Speech | Detects neuro change |
| Vital signs | Watch for Cushing’s triad |
| Respiratory pattern | Brainstem/oxygenation concern |
| Headache | May worsen with ICP |
| Vomiting | Can occur with increased pressure |
| Seizure activity | Brain irritation/injury concern |
| Intake and output | Fluid balance and medication effects |
| Drainage from ears or nose | Possible CSF leak after trauma |
| Bilateral neuro assessment | Compare sides for changes |

## Report Immediately

| Red Flag | Why It Is Serious |
| -------- | ----------------- |
| Change in LOC or new confusion | Earliest ICP warning |
| Unequal, fixed, or dilated pupils | Possible worsening brain pressure |
| New weakness | New neuro deficit |
| Seizure | Brain irritation/injury |
| Cushing’s triad | Late emergency sign |
| Projectile vomiting | Increased pressure concern |
| Worsening headache | Worsening ICP concern |
| Abnormal posturing | Severe neuro deterioration |
| Fever | Increases metabolic demand / possible infection or hypothalamus injury |
| Drainage from ears or nose | Possible CSF leak |

---

# Nursing Interventions for Increased ICP

## Intervention Chart

| Intervention | Nursing Action | Why |
| ------------ | -------------- | --- |
| Positioning | Semi-Fowler’s position | Promotes venous drainage |
| Head alignment | Keep head neutral and midline | Prevents blocked venous return |
| Neck positioning | Avoid neck flexion/rotation | Promotes drainage from brain |
| Hip positioning | Avoid extreme hip flexion | Helps venous return |
| Neuro checks | Monitor LOC closely and compare bilaterally | Detects worsening early |
| Vital signs | Monitor carefully | Detects Cushing’s triad |
| Environment | Keep calm, limit stimulation, allow rest | Prevents ICP spikes |
| Safety | Treat as spinal injury until ruled out | Prevents further injury |
| CSF leak check | Monitor for halo sign if fluid leaks from ears or nose | Possible basilar skull fracture/CSF leak |
| Oxygenation | Monitor respiratory status and ABGs | Prevents secondary brain injury |
| Oxygen goal | Keep PaO₂ above 80 mm Hg as ordered | Supports brain oxygenation |
| Escalation | Prepare for advanced oxygen therapy if needed | Airway/ventilation support |

<div class="nclex-box">
<strong>NCLEX Tip:</strong> Do not cluster care with increased ICP. Too much activity at once can increase stimulation and raise ICP.
</div>

---

# Fever and Increased ICP

| Fever Concern | Why It Matters |
| ------------- | -------------- |
| Fever after neuro injury | Can increase brain oxygen demand |
| Infection | Possible cause of fever |
| Hypothalamus injury | Can affect temperature regulation |
| Nursing action | Report fever immediately |

---

# What To Avoid With Increased ICP

| Avoid | Why |
| ----- | --- |
| Coughing | Increases pressure |
| Straining | Increases pressure |
| Valsalva maneuver | Increases pressure |
| Excessive suctioning | Can increase ICP |
| Prolonged suctioning | Can decrease oxygen and increase pressure |
| Overstimulation | Can worsen ICP |
| Hip flexion | Reduces venous drainage |
| Neck flexion/rotation | Reduces venous drainage |
| Tight collars or ties | Can restrict venous return |

---

# Medications and Treatments for Increased ICP

| Treatment | Purpose | Monitor / Nursing Notes |
| --------- | ------- | ----------------------- |
| Steroids | May decrease inflammation and swelling depending on cause | Monitor glucose, infection risk, provider orders |
| Anticonvulsants | Prevent or treat seizures | Examples: phenytoin/Dilantin, phenobarbital |
| Mannitol | Osmotic diuretic; pulls fluid from brain into bloodstream so kidneys can remove it | Urine output, lung sounds, BP, electrolytes, kidney function, dehydration |
| Furosemide/Lasix | Diuretic; may help remove fluid | Potassium, BP, urine output, hydration status |
| Hypertonic saline | Helps reduce ICP by pulling fluid from brain tissue | Sodium level, neuro status, fluid status, lung sounds, IV site |
| Opioids | Fentanyl may be used for ventilated clients to control pain/decrease agitation | Respiratory status, sedation level, BP |
| Barbiturate coma | May be used when other interventions are unsuccessful | ICU-level monitoring |

<div class="tip-box">
<strong>Priority Tip:</strong> Mannitol and hypertonic saline both affect fluid balance. Watch lung sounds and neuro status so the client does not worsen from fluid overload or electrolyte changes.
</div>

---

# Measuring ICP

| Device | Basic Point | NCLEX Cue |
| ------ | ----------- | --------- |
| Intraventricular catheter | Placed in ventricle | Highest infection risk |
| Subarachnoid bolt or screw | Measures pressure in one spot only | Less flexible than ventricular catheter |
| Epidural sensor/transducer | Indirect measurement | Less accurate |
| Parenchymal fiberoptic catheter | Very accurate and reliable but expensive | High accuracy |

---

# Hydrocephalus

## Big Picture

| Concept | Simple Explanation |
| ------- | ------------------ |
| Hydrocephalus | Buildup of CSF in the brain |
| Main problem | Excess fluid increases pressure on brain tissue |
| Goal of treatment | Remove extra fluid and decrease pressure |
| Common treatment | Surgically placed shunt |

---

# Basic CSF Flow

| Structure / Flow | What To Know |
| ---------------- | ------------ |
| Lateral ventricles | CSF is made here |
| Third ventricle | CSF flows through connected spaces |
| Fourth ventricle | CSF continues toward brain/spinal cord spaces |
| Brain and spinal cord | CSF bathes and cushions them |
| Reabsorption | CSF is normally reabsorbed |

## What Goes Wrong

| Problem | Result |
| ------- | ------ |
| CSF flow is blocked | CSF backs up |
| CSF is not absorbed properly | CSF builds up |
| Too much CSF builds up | Pressure increases |
| Narrowing/blockage of aqueduct of Sylvius | Common cause of obstruction |

---

# Hydrocephalus Manifestations

## Infants vs Older Children/Adults

| Infants | Older Children and Adults |
| ------- | ------------------------- |
| Enlarged head circumference | Headache |
| Bulging fontanel | Impaired vision |
| Tense fontanel | Cognitive difficulties |
| High-pitched or shrill cry | Loss of coordination |
| Poor feeding | Incontinence |
| Vomiting | Nausea and vomiting |
| Irritability | Confusion |
| Sleepiness | Decreased LOC |
| Seizures | Seizures |
| Sunsetting eyes | Motor/coordination changes |
| Developmental delay | Gait or balance problems |

<div class="nclex-box">
<strong>NCLEX Tip:</strong> In infants, the skull bones are not fully fused, so increased CSF may show up as an enlarged head circumference and bulging fontanel.
</div>

---

# Hydrocephalus Priority Assessment

| Assess | Why It Matters |
| ------ | -------------- |
| Level of consciousness | Shows neuro status |
| Head circumference in infants | Tracks increasing CSF/pressure |
| Fontanels in infants | Bulging/tense fontanel suggests pressure |
| Feeding tolerance | Poor feeding can signal worsening |
| Vomiting | Increased pressure/shunt concern |
| Eye movement | Sunsetting eyes or pupil changes |
| Pupils | Neuro status |
| Seizure activity | Worsening brain irritation |
| Motor function | Neuro changes |
| Coordination | Function and neuro status |
| Vital signs | Detect worsening condition |

## Report Immediately

| Red Flag | Think |
| -------- | ----- |
| Bulging fontanel | Increased pressure |
| Rapid increase in head size | Worsening hydrocephalus |
| High-pitched cry | Neuro irritation/increased pressure |
| Sunsetting eyes | Hydrocephalus sign |
| Vomiting | Increased pressure/shunt concern |
| Seizure | Neuro emergency |
| Decreased LOC | Worsening neuro status |
| Signs of shunt infection or malfunction | Shunt complication |

---

# Hydrocephalus Treatment

| Treatment Goal | How It Helps |
| -------------- | ------------ |
| Remove excess fluid | Decreases pressure on brain tissue |
| Decrease ICP | Protects brain perfusion |
| Shunt placement | Drains CSF to another body area |

---

# Shunt Types

| Shunt | Full Name | Pathway | Simple Meaning |
| ----- | --------- | ------- | -------------- |
| VP shunt | Ventriculoperitoneal shunt | Brain ventricle → tubing under skin → peritoneal cavity | Drains CSF from brain to abdomen |
| VA shunt | Ventriculoatrial shunt | Brain ventricle → tubing under skin → vascular system/heart area | Drains CSF from brain to blood/heart area |

---

# Nursing Care After Shunt Placement

| Priority Care | Why |
| ------------- | --- |
| Monitor vital signs | Detect infection or deterioration |
| Monitor neurologic status | Detect increased ICP or shunt problems |
| Assess incision sites | Watch for infection/drainage |
| Monitor for infection | Shunt infection can be serious |
| Monitor for pain | Comfort and complication clue |
| Administer antibiotics as ordered | Prevent/treat infection |
| Administer pain medication as ordered | Comfort and decreased agitation |
| Measure head circumference in infants | Tracks CSF/pressure changes |
| Assess fontanels in infants | Detects increased pressure |
| Teach family dressing care as ordered | Prevent infection |
| Teach family signs of shunt problems | Early recognition at home |

---

# Shunt Positioning Safety

| Do Not | Why |
| ------ | --- |
| Do not place pressure on the shunt side | Prevents skin breakdown and shunt compression |
| Do not position infant directly on the side of the shunt unless ordered | Prevents pressure on shunt apparatus |
| Do not sit upright immediately after shunt placement unless ordered | May drain CSF too quickly depending on orders |

<div class="tip-box">
<strong>Priority Tip:</strong> After shunt placement, follow the provider’s positioning order. NCLEX may test that you should not place pressure on the shunt side.
</div>

---

# Shunt Malfunction vs Shunt Infection

| Shunt Malfunction | Shunt Infection |
| ----------------- | --------------- |
| High-pitched cry | Fever |
| Irritability | Redness at incision |
| Vomiting | Swelling |
| Poor feeding | Drainage |
| Sleepiness | Irritability |
| Bulging fontanel | Neck stiffness |
| Increased head circumference | Abdominal pain |
| Headache | Poor feeding |
| Seizures | Possible worsening neuro signs |
| Decreased LOC | Possible sepsis concern |
| Return of hydrocephalus symptoms | Incision or systemic infection signs |

# Quick NCLEX Review Chart

| Topic | What To Remember |
| ----- | ---------------- |
| Increased ICP | Too much pressure inside rigid skull |
| Skull contents | Brain tissue, CSF, blood |
| Earliest ICP sign | Change in LOC |
| Most important assessment | LOC |
| Normal ICP | 0–15 mm Hg |
| Life-threatening ICP | 20 mm Hg or higher for more than 5 minutes |
| Cushing’s triad | Increased systolic BP/widened pulse pressure, bradycardia, respiratory depression |
| ICP vs shock | Opposite vital sign pattern |
| ICP positioning | Semi-Fowler’s, head neutral midline |
| Avoid with ICP | Straining, coughing, excessive suctioning, overstimulation |
| Fever with ICP | Report immediately; increases oxygen demand or may signal hypothalamus/infection issue |

# Most Important “What Do I Do First?” Points

| Situation | First / Priority Thought |
| --------- | ------------------------ |
| Suspected increased ICP | Assess LOC and neuro status |
| Decreased LOC | Airway and oxygenation become priority |
| Cushing’s triad | Late emergency sign — report/intervene quickly |
| Drainage from ears or nose after trauma | Check for halo sign; possible CSF leak |
| Increased ICP positioning question | Semi-Fowler’s, head neutral/midline |
| ICP care planning | Avoid coughing, straining, Valsalva, overstimulation, excessive suctioning |
| Infant with bulging fontanel and high-pitched cry | Think hydrocephalus/increased ICP |
| Shunt client with vomiting or decreased LOC | Think shunt malfunction |
| Shunt client with fever/red incision/drainage | Think shunt infection |
