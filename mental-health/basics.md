# Mental Health Basics

<div class="note">
Mental health nursing is less about “fixing” the patient’s problem in one conversation and more about <strong>safety, trust, therapeutic communication, boundaries, and helping the patient use healthy coping skills.</strong>
</div>

---

## Big Picture

Mental health nursing focuses on:

| Main Priority                 | What It Means                                                              |
| ----------------------------- | -------------------------------------------------------------------------- |
| **Safety first**              | Protect the patient and others from harm                                   |
| **Therapeutic communication** | Use calm, supportive, nonjudgmental communication                          |
| **Reality-based care**        | Do not argue with delusions, but do not agree with them                    |
| **Least restrictive care**    | Use the safest option that limits freedom the least                        |
| **Trust and boundaries**      | Be consistent, honest, and professional                                    |
| **Coping support**            | Help the patient use healthy ways to manage stress, emotions, and symptoms |

---

## Nursing Priorities in Mental Health

### Priority Order

1. **Safety**

   * Suicide risk
   * Homicide risk
   * Self-harm
   * Violence/aggression
   * Severe withdrawal
   * Severe panic or loss of control

2. **Physiological needs**

   * Sleep
   * Food/fluid intake
   * Medication effects
   * Withdrawal symptoms
   * Medical complications

3. **Therapeutic communication**

   * Stay calm
   * Build trust
   * Encourage expression
   * Use simple, clear statements

4. **Coping and education**

   * Teach coping skills
   * Encourage support systems
   * Reinforce medication teaching
   * Help with discharge planning

## Therapeutic Communication

Therapeutic communication helps the patient feel heard, safe, and respected.

### Use These

| Therapeutic Technique       | Example                                                                     |
| --------------------------- | --------------------------------------------------------------------------- |
| **Open-ended questions**    | “Can you tell me more about what you’re feeling?”                           |
| **Restating**               | “You feel overwhelmed right now.”                                           |
| **Reflection**              | “What do you think would help you feel safer?”                              |
| **Offering self**           | “I can stay with you for a while.”                                          |
| **Silence**                 | Sitting quietly without rushing the patient                                 |
| **Clarification**           | “When you say you feel unsafe, what do you mean?”                           |
| **Presenting reality**      | “I do not see anyone in the room, but I understand this feels real to you.” |
| **Focusing**                | “Let’s talk more about what happened before you felt this way.”             |
| **Encouraging description** | “What does the anxiety feel like in your body?”                             |

---

## Nontherapeutic Communication

These responses can make the patient feel judged, dismissed, or unsafe.

| Avoid This                            | Why                                                    |
| ------------------------------------- | ------------------------------------------------------ |
| **“Why did you do that?”**            | Sounds blaming                                         |
| **False reassurance**                 | “Everything will be fine” may feel dismissive          |
| **Giving advice**                     | The nurse should guide, not decide for the patient     |
| **Changing the subject**              | Avoids the patient’s concern                           |
| **Approval/disapproval**              | The nurse should not judge behavior as “good” or “bad” |
| **Arguing**                           | Increases defensiveness and does not help delusions    |
| **Asking too many questions at once** | Can overwhelm the patient                              |
| **Promising secrecy**                 | Safety concerns must be reported                       |

<div class="warning">
<strong>NCLEX:</strong> Never promise to keep abuse, suicide plans, homicide threats, or safety concerns secret.
</div>

## Mental Status Assessment

The mental status assessment helps the nurse understand the patient’s emotional, cognitive, and behavioral state.

| Area                | What the Nurse Assesses                                    |
| ------------------- | ---------------------------------------------------------- |
| **Appearance**      | Hygiene, clothing, posture, eye contact                    |
| **Behavior**        | Calm, restless, withdrawn, aggressive, cooperative         |
| **Speech**          | Fast, slow, pressured, soft, loud, delayed                 |
| **Mood**            | Patient’s stated emotional state                           |
| **Affect**          | Emotional expression the nurse observes                    |
| **Thought process** | Organized, racing, loose associations, flight of ideas     |
| **Thought content** | Delusions, paranoia, suicidal thoughts, homicidal thoughts |
| **Perception**      | Hallucinations or illusions                                |
| **Cognition**       | Orientation, memory, attention                             |
| **Insight**         | Does the patient understand their condition?               |
| **Judgment**        | Can the patient make safe decisions?                       |

## Mood vs Affect

| Term       | Meaning                        | Example                                    |
| ---------- | ------------------------------ | ------------------------------------------ |
| **Mood**   | How the patient says they feel | “I feel sad.”                              |
| **Affect** | What the nurse observes        | Patient is crying, flat, angry, or smiling |

### Affect Terms

| Affect                   | Meaning                               |
| ------------------------ | ------------------------------------- |
| **Flat affect**          | Little or no emotional expression     |
| **Blunted affect**       | Decreased emotional expression        |
| **Labile affect**        | Rapidly changing emotions             |
| **Inappropriate affect** | Emotion does not match the situation  |
| **Constricted affect**   | Limited range of emotional expression |

---

## Common Thought Process Terms

| Term                      | Meaning                                                |
| ------------------------- | ------------------------------------------------------ |
| **Flight of ideas**       | Rapidly jumping from one idea to another               |
| **Loose associations**    | Thoughts are not clearly connected                     |
| **Tangential speech**     | Goes off topic and does not return to the point        |
| **Circumstantial speech** | Gives too many details but eventually answers          |
| **Thought blocking**      | Suddenly stops speaking because thought is interrupted |
| **Perseveration**         | Repeats the same word, thought, or idea                |
| **Word salad**            | Words are mixed together and do not make sense         |

---

## Hallucinations vs Delusions

| Term              | Meaning                                    | Nursing Response                           |
| ----------------- | ------------------------------------------ | ------------------------------------------ |
| **Hallucination** | Sensory perception without a real stimulus | Assess safety and present reality          |
| **Delusion**      | Fixed false belief                         | Do not argue; focus on feelings and safety |
| **Illusion**      | Misinterpretation of something real        | Clarify and reduce anxiety                 |

### Hallucination Example

Patient says:
“I hear voices telling me I am worthless.”

Best nursing response:
“I do not hear the voices, but I understand they are upsetting. Are the voices telling you to hurt yourself or anyone else?”

<div class="note">
<strong>NCLEX:</strong> For hallucinations, always assess if the voices are command hallucinations telling the patient to harm self or others.
</div>

---

## Therapeutic Response to Delusions

Do **not** agree with the delusion.

Do **not** argue with the delusion.

Focus on:

* The patient’s feelings
* Safety
* Reality-based statements
* Redirection

| Patient Statement         | Best Nurse Response                                                                          |
| ------------------------- | -------------------------------------------------------------------------------------------- |
| “The FBI is watching me.” | “That sounds frightening. I do not see anyone watching you, but I can stay with you.”        |
| “My food is poisoned.”    | “You seem worried about the food. Let’s talk about what would help you feel safe eating.”    |
| “I am the president.”     | “I understand that you believe that. Right now, we are in the hospital and I am your nurse.” |

## Defense Mechanisms

Defense mechanisms are unconscious ways people protect themselves from stress, anxiety, or uncomfortable feelings.

| Defense Mechanism      | Meaning                                     | Example                                           |
| ---------------------- | ------------------------------------------- | ------------------------------------------------- |
| **Denial**             | Refusing to accept reality                  | “I do not have a drinking problem.”               |
| **Projection**         | Blaming feelings on someone else            | “My nurse hates me” when the patient is angry     |
| **Displacement**       | Taking feelings out on a safer target       | Yelling at family after being upset with provider |
| **Regression**         | Acting younger during stress                | Adult crying or becoming dependent                |
| **Rationalization**    | Making excuses                              | “I failed because the teacher hates me.”          |
| **Repression**         | Unconsciously blocking memory               | Cannot remember traumatic event                   |
| **Suppression**        | Consciously choosing not to think about it  | “I’ll deal with this after my exam.”              |
| **Sublimation**        | Channeling feelings into something positive | Using exercise to manage anger                    |
| **Reaction formation** | Acting opposite of true feelings            | Being overly nice to someone you dislike          |
| **Splitting**          | Seeing people as all good or all bad        | “This nurse is perfect, but that nurse is evil.”  |

<div class="tip">
<strong>Best defense mechanism:</strong> Sublimation is usually considered a healthy defense mechanism because the person channels stress into a safe activity.
</div>

## Boundaries

Boundaries protect both the nurse and the patient.

### Healthy Nursing Boundaries

| Do                      | Avoid                                 |
| ----------------------- | ------------------------------------- |
| Be consistent           | Making special exceptions             |
| Set clear limits        | Arguing or threatening                |
| Keep promises realistic | Promising secrecy                     |
| Stay professional       | Sharing too much personal information |
| Follow the care plan    | Letting the patient manipulate staff  |
| Use team communication  | Allowing splitting between staff      |

### Limit-Setting Example

Patient says:
“If you really cared, you would let me stay up all night.”

Therapeutic response:
“I understand you are upset. The rule is that lights are out at 10 PM. You may read quietly in bed or use a relaxation technique.”

## Milieu Therapy

Milieu therapy means the environment is part of the treatment.

The goal is to provide a safe, structured, supportive setting.

| Milieu Therapy Includes       |
| ----------------------------- |
| Clear routines                |
| Group therapy                 |
| Patient responsibilities      |
| Safe environment              |
| Consistent staff expectations |
| Therapeutic activities        |
| Positive coping skills        |
| Respectful communication      |

### Nursing Role in Milieu Therapy

The nurse:

* Maintains safety
* Encourages participation
* Sets limits
* Models healthy communication
* Reinforces rules consistently
* Observes patient interactions
* Reduces environmental triggers

## Voluntary vs Involuntary Admission

| Type                      | Meaning                                                                                                |
| ------------------------- | ------------------------------------------------------------------------------------------------------ |
| **Voluntary admission**   | Patient agrees to treatment                                                                            |
| **Involuntary admission** | Patient is admitted because they are a danger to self, danger to others, or unable to meet basic needs |

### Important Point

A patient can refuse some treatments unless there is an emergency or legal order.

However, safety always comes first.

## Patient Rights

Mental health patients still have rights.

| Patient Right                   | Meaning                                          |
| ------------------------------- | ------------------------------------------------ |
| Right to privacy                | Personal information is protected                |
| Right to dignity                | Patient should be treated respectfully           |
| Right to refuse treatment       | Unless emergency/legal order applies             |
| Right to least restrictive care | Use the safest option with the least restriction |
| Right to informed consent       | Patient should understand treatment              |
| Right to communication          | Phone/mail/visitors unless restricted for safety |
| Right to safe care              | Environment should protect patient and others    |

## Restraints and Seclusion

Restraints and seclusion are used only when less restrictive interventions fail and the patient is at risk of harming self or others.

They are **not** used for:

* Punishment
* Convenience
* Staff frustration
* Lack of staffing

### Before Restraints or Seclusion

Try:

* Calm communication
* Reduce stimulation
* Offer quiet space
* Remove audience
* Set limits
* Offer medication if ordered
* Call for help early

### Nursing Care During Restraints

| Nursing Action         | Why It Matters               |
| ---------------------- | ---------------------------- |
| Assess circulation     | Prevent injury               |
| Assess breathing       | Make sure airway is safe     |
| Assess skin            | Prevent breakdown            |
| Check vital signs      | Monitor physical status      |
| Offer fluids/toileting | Meet basic needs             |
| Reassess behavior      | Remove restraints ASAP       |
| Document clearly       | Legal and safety requirement |

<div class="warning">
<strong>NCLEX:</strong> Restraints are always a last resort and must be discontinued as soon as the patient is safe.
</div>

## Suicide Assessment Basics

Suicide risk is always a priority.

Ask directly. Talking about suicide does **not** put the idea in the patient’s head.

### Suicide Questions

| Assessment Question                        | Why It Matters                  |
| ------------------------------------------ | ------------------------------- |
| “Are you thinking about hurting yourself?” | Determines suicidal ideation    |
| “Do you have a plan?”                      | Plan increases risk             |
| “Do you have access to the method?”        | Access increases danger         |
| “When were you planning to do this?”       | Immediacy increases priority    |
| “Have you tried before?”                   | Previous attempt increases risk |
| “What has stopped you so far?”             | Identifies protective factors   |

### Highest Risk

The patient is highest risk when they have:

* Specific plan
* Access to means
* Time chosen
* Previous attempt
* Giving away belongings
* Sudden calmness after depression
* Command hallucinations
* Substance use
* Hopelessness

### Nursing Priority

If a patient is actively suicidal:

1. **Stay with the patient**
2. **Remove harmful objects**
3. **Notify the provider/team**
4. **Initiate suicide precautions**
5. **Document assessment and interventions**

## Violence Risk Basics

Warning signs may include:

| Warning Sign        |
| ------------------- |
| Pacing              |
| Clenched fists      |
| Loud voice          |
| Threats             |
| Agitation           |
| Staring             |
| History of violence |
| Paranoia            |
| Substance use       |
| Escalating anxiety  |

### Nursing Interventions

| Do                          | Why                             |
| --------------------------- | ------------------------------- |
| Stay calm                   | Helps reduce escalation         |
| Keep exit accessible        | Protects nurse safety           |
| Use short, clear statements | Prevents overwhelm              |
| Give choices                | Helps patient feel some control |
| Reduce stimulation          | Decreases agitation             |
| Call for help early         | Prevents injury                 |
| Avoid touching patient      | Touch may increase aggression   |

---

## Anxiety Levels

| Level        | Signs                                          | Nursing Approach                          |
| ------------ | ---------------------------------------------- | ----------------------------------------- |
| **Mild**     | Alert, focused, slightly restless              | Teach and problem-solve                   |
| **Moderate** | Narrowed focus, increased HR, tense            | Use simple directions                     |
| **Severe**   | Poor concentration, pacing, physical symptoms  | Reduce stimuli, stay calm                 |
| **Panic**    | Feels out of control, may be unable to process | Stay with patient, short commands, safety |

<div class="note">
<strong>NCLEX:</strong> Do not try to teach during severe anxiety or panic. First reduce anxiety and keep the patient safe.
</div>

---

## Panic Attack Nursing Care

During a panic attack:

| Nursing Action            | Why                                      |
| ------------------------- | ---------------------------------------- |
| Stay with the patient     | Reduces fear and promotes safety         |
| Use calm voice            | Helps decrease escalation                |
| Give short directions     | Patient may not process complex teaching |
| Reduce stimuli            | Lowers anxiety                           |
| Encourage slow breathing  | Helps with hyperventilation              |
| Avoid crowded areas       | Too much stimulation worsens panic       |
| Teach coping skills later | Teaching is best after panic decreases   |

Example response:
“You are safe. Stay with me. Take a slow breath in and out.”

## Grief Basics

Grief is a normal response to loss.

| Stage          | Meaning                              |
| -------------- | ------------------------------------ |
| **Denial**     | “This cannot be happening.”          |
| **Anger**      | “Why did this happen?”               |
| **Bargaining** | “If this changes, I’ll do anything.” |
| **Depression** | Sadness and withdrawal               |
| **Acceptance** | Begins to adjust to reality          |

### Nursing Care for Grief

* Let the patient express feelings
* Avoid false reassurance
* Do not rush the grieving process
* Encourage support systems
* Stay present
* Use silence when appropriate

Best response:
“This is very painful. I can sit with you.”

## Medication Safety Basics

Mental health medications can take time to work and may have serious side effects.

| Medication Group       | Nursing Focus                                   |
| ---------------------- | ----------------------------------------------- |
| **Antidepressants**    | Suicide risk may increase early in treatment    |
| **Benzodiazepines**    | Sedation, falls, dependence                     |
| **Mood stabilizers**   | Toxicity and lab monitoring                     |
| **Antipsychotics**     | EPS, tardive dyskinesia, NMS, metabolic effects |
| **Stimulants**         | Appetite loss, insomnia, increased HR/BP        |
| **Substance-use meds** | Withdrawal safety and relapse prevention        |

---

## Common Mental Health Medication Emergencies

| Emergency                          | Clues                                                               | Priority                          |
| ---------------------------------- | ------------------------------------------------------------------- | --------------------------------- |
| **Serotonin syndrome**             | Fever, agitation, sweating, diarrhea, tremors, hyperreflexia        | Stop medication, notify provider  |
| **Neuroleptic malignant syndrome** | High fever, muscle rigidity, altered mental status, unstable vitals | Medical emergency                 |
| **Lithium toxicity**               | GI upset, tremor, confusion, ataxia, seizures                       | Hold lithium, notify provider     |
| **Benzodiazepine overdose**        | Respiratory depression, sedation                                    | Airway and breathing priority     |
| **Alcohol withdrawal/DTs**         | Tremors, agitation, hallucinations, seizures                        | Safety, seizure precautions, meds |

## NCLEX Therapeutic Communication Rules

| NCLEX Rule                    | Remember                                           |
| ----------------------------- | -------------------------------------------------- |
| Do not ask “why”              | It sounds blaming                                  |
| Do not give false reassurance | Avoid “everything will be okay”                    |
| Do not give advice            | Help patient explore choices                       |
| Do not agree with delusions   | Present reality calmly                             |
| Do not argue                  | Arguing increases defensiveness                    |
| Do not promise secrecy        | Safety must be reported                            |
| Use open-ended questions      | Encourages expression                              |
| Focus on feelings             | Especially with delusions/anxiety                  |
| Use silence                   | Allows patient time to process                     |
| Set clear limits              | Especially with manipulation, aggression, or mania |

## NCLEX Priority Cues

| Situation                          | Priority Nursing Action                                  |
| ---------------------------------- | -------------------------------------------------------- |
| Patient is suicidal                | Stay with patient and remove harmful objects             |
| Patient has command hallucinations | Ask what the voices are saying                           |
| Patient is violent/agitated        | Maintain safety, use calm limit-setting, call for help   |
| Patient is having a panic attack   | Stay with patient, reduce stimuli, use short directions  |
| Patient has delusions              | Do not argue; present reality and focus on feelings      |
| Patient is manic                   | Decrease stimulation, set limits, provide finger foods   |
| Patient refuses medication         | Assess reason, educate, notify provider if needed        |
| Patient is splitting staff         | Maintain consistent boundaries and team communication    |
| Patient is grieving                | Allow expression and use therapeutic presence            |
| Patient is restrained              | Assess safety, circulation, skin, needs, and remove ASAP |

| Patient Says                                 | Best Response                                                                      |
| -------------------------------------------- | ---------------------------------------------------------------------------------- |
| “I want to die.”                             | “Are you thinking about killing yourself?”                                         |
| “The voices are telling me things.”          | “What are the voices telling you?”                                                 |

| “Why did this happen to me?”                 | “This feels very unfair and painful.”                                              |
| “I’m so angry I could hit someone.”          | “I can see you are angry. I will help you stay safe.”                              |
