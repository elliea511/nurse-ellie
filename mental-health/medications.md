# Mental Health Medications

<div class="note">

Mental health medication questions usually test **safety, side effects, toxicity, patient teaching, and when to notify the provider.**

For NCLEX, focus on: **suicide risk, lithium toxicity, serotonin syndrome, NMS, EPS, benzodiazepine safety, withdrawal medications, and medication adherence.**

</div>

---

## Big Picture

| Medication Group | Used For | Big Nursing Focus |
|---|---|---|
| **Antidepressants** | Depression, anxiety, OCD, PTSD | Suicide risk early, serotonin syndrome |
| **Antianxiety meds** | Anxiety, panic, agitation | Sedation, falls, dependence |
| **Mood stabilizers** | Bipolar disorder, mood swings | Lithium toxicity, labs |
| **Antipsychotics** | Schizophrenia, psychosis, mania | EPS, NMS, metabolic effects |
| **Substance-use meds** | Overdose, withdrawal, relapse prevention | Respirations, seizures, alcohol avoidance |
| **Stimulants** | ADHD, some binge eating treatment | Appetite, sleep, HR/BP |

<div class="warning">

**NCLEX:** Mental health meds can take time to work, but dangerous side effects can happen early. Always know what must be reported immediately.

</div>

# Antidepressants

Antidepressants are used for: major depressive disorder, persistent depressive disorder, anxiety disorders, OCD, PTSD, PMDD, and some somatic disorders.

---

## Antidepressant Classes

| Class | Examples | Nursing Focus |
|---|---|---|
| **SSRIs** | citalopram, escitalopram, fluoxetine, sertraline | Suicide risk early, serotonin syndrome |
| **SNRIs** | venlafaxine, duloxetine | BP, serotonin syndrome |
| **NDRI** | bupropion / Wellbutrin | Seizure risk |
| **TCAs** | imipramine, amitriptyline | Anticholinergic effects, overdose danger |
| **MAOIs** | phenelzine, tranylcypromine | Tyramine diet, hypertensive crisis |
| **Herbal** | St. John's Wort | Interactions, serotonin syndrome risk |

---

## General Antidepressant Teaching

| Teaching | Why |
|---|---|
| Takes several weeks to work | Patient may stop too early |
| Take as prescribed | Prevents relapse |
| Do not stop suddenly | Withdrawal/relapse risk |
| Report suicidal thoughts | Safety priority |
| Avoid alcohol | CNS effects/safety risk |
| Tell provider about herbals | St. John's Wort has interactions |
| Watch for serotonin syndrome | Can be life-threatening |

<div class="warning">

**NCLEX:** Antidepressants may increase energy before mood improves. This can increase suicide risk early in treatment.

</div>

---

## Serotonin Syndrome

Serotonin syndrome can happen when there is too much serotonin. Higher risk with SSRIs, SNRIs, MAOIs, St. John's Wort, or multiple serotonergic meds together.

| Signs/Symptoms |
|---|
| Fever |
| Agitation |
| Confusion |
| Sweating |
| Diarrhea |
| Tremors |
| Hyperreflexia |
| Muscle rigidity |
| Tachycardia |
| Hypertension |

<div class="warning">

**Priority:** Hold the medication and notify the provider if serotonin syndrome is suspected.

</div>

# SSRIs and SNRIs

## SSRIs

SSRIs increase serotonin availability. Examples: citalopram, escitalopram, fluoxetine, sertraline, paroxetine.

## Nursing Focus

| Concern | Teaching |
|---|---|
| Delayed effect | Takes weeks |
| Suicide risk early | Report worsening mood or suicidal thoughts |
| GI upset | May occur early |
| Sexual side effects | Report if bothersome |
| Serotonin syndrome | Report fever, agitation, diarrhea, tremor |
| Stopping suddenly | Do not abruptly discontinue |

---

## SNRIs

SNRIs affect serotonin and norepinephrine. Examples: venlafaxine, duloxetine.

| Nursing Focus |
|---|
| Monitor BP |
| Watch for serotonin syndrome |
| Do not stop suddenly |
| Takes weeks to work |
| Report suicidal thoughts |

---

# Bupropion / Wellbutrin

Bupropion is an NDRI. Used for depression, smoking cessation support, and sometimes other conditions.

| Nursing Focus |
|---|
| Can lower seizure threshold |
| Avoid or use caution with seizure history |
| Avoid or use caution with eating disorder history |
| May cause insomnia |
| Monitor mood/suicide risk |

<div class="note">

Bupropion is a common NCLEX safety trap because of seizure risk.

</div>

---

# TCAs

Tricyclic antidepressants include imipramine and amitriptyline.

## Side Effects

| Side Effect Type | Examples |
|---|---|
| Anticholinergic | Dry mouth, constipation, urinary retention, blurred vision |
| Cardiovascular | Orthostatic hypotension, dysrhythmias |
| CNS | Sedation, dizziness |
| Safety | Dangerous in overdose |

## Teaching

| Teaching |
|---|
| Change positions slowly |
| Increase fluids/fiber if allowed |
| Use sugar-free candy for dry mouth |
| Avoid alcohol/CNS depressants |
| Keep medication away from suicidal patients |
| Report chest pain, palpitations, severe dizziness |

---

# MAOIs

MAOIs include phenelzine and tranylcypromine.

## Big Risk

MAOIs can cause **hypertensive crisis** if taken with tyramine-rich foods.

## Avoid Tyramine Foods

| Avoid |
|---|
| Aged cheese |
| Cured meats |
| Smoked meats |
| Fermented foods |
| Soy sauce |
| Sauerkraut |
| Red wine |
| Draft beer |
| Some overripe foods |

## Hypertensive Crisis Symptoms

| Symptoms |
|---|
| Severe headache |
| Stiff neck |
| Hypertension |
| Chest pain |
| Palpitations |
| Nausea/vomiting |
| Sweating |
| Confusion |

<div class="warning">

**NCLEX:** MAOI + tyramine = hypertensive crisis risk.

</div>

# Antianxiety Medications

| Medication Type | Examples | Key Point |
|---|---|---|
| **Benzodiazepines** | lorazepam, diazepam, alprazolam | Fast relief, dependence risk |
| **Buspirone** | buspirone / BuSpar | Long-term anxiety, takes weeks |
| **Antihistamines** | hydroxyzine | Sedating |
| **Beta blockers** | propranolol | Helps physical anxiety symptoms |

# Benzodiazepines

Benzodiazepines provide quick anxiety relief. Also used for alcohol withdrawal, severe agitation, and seizure prevention/treatment.

Examples: diazepam (Valium), lorazepam (Ativan), chlordiazepoxide (Librium).

## Nursing Focus

| Concern | Why |
|---|---|
| Sedation | Fall risk |
| Respiratory depression | Especially with opioids/alcohol |
| Dependence | Short-term use preferred |
| Tolerance | Higher doses may be needed over time |
| Withdrawal seizures | Do not stop suddenly |
| Confusion | Higher risk in older adults |

## Teaching

| Teaching |
|---|
| Avoid alcohol |
| Avoid opioids/CNS depressants unless prescribed and monitored |
| Do not drive until effects are known |
| Rise slowly |
| Do not stop suddenly after long-term use |
| Use only as prescribed |

<div class="warning">

Benzodiazepines plus alcohol or opioids can cause dangerous respiratory depression.

</div>

---

# Buspirone

Buspirone is used for long-term anxiety management.

| Key Teaching |
|---|
| Takes weeks to work |
| Not for immediate panic relief |
| No tolerance/addiction like benzodiazepines |
| Take consistently |
| Avoid use with severe liver/kidney disease unless provider approves |

<div class="note">

**Buspirone is not a rescue medication.** It will not stop an active panic attack right away.

</div>

---

# Beta Blockers for Anxiety

Propranolol may be used to reduce physical symptoms of anxiety.

| Helps With |
|---|
| Tremors |
| Tachycardia |
| Shaky voice |
| Performance anxiety symptoms |

## Nursing Focus

| Monitor |
|---|
| Heart rate |
| Blood pressure |
| Dizziness |
| Bronchospasm risk in asthma/COPD depending on medication |

# Mood Stabilizers

Mood stabilizers are used mainly for bipolar disorder.

| Medication Type | Examples |
|---|---|
| Lithium | Lithium carbonate |
| Anticonvulsants | valproate, carbamazepine, lamotrigine |
| Antipsychotics | May be used for mania or psychotic features |

---

# Lithium

Lithium is a classic drug of choice for bipolar disorder. Lithium is a **salt**, so sodium and fluid balance matter.

## Therapeutic Level

| Level | Meaning |
|---|---|
| **0.6–1.2 mEq/L** | Therapeutic range |
| Above range | Toxicity risk |

---

## Lithium Toxicity

| Early Signs | Severe Signs |
|---|---|
| Nausea/vomiting | Confusion |
| Diarrhea | Ataxia |
| Drowsiness | Severe tremor |
| Muscle weakness | Seizures |
| Fine tremor worsening | Coma |
| Slurred speech | Dysrhythmias |

<div class="warning">

**If lithium toxicity is suspected:** hold the dose and notify the provider.

</div>

---

## Lithium Teaching

| Teaching | Why |
|---|---|
| Maintain consistent salt intake | Low sodium increases lithium retention |
| Drink adequate fluids | Dehydration increases toxicity |
| Avoid dehydration | Vomiting, diarrhea, sweating increase risk |
| Avoid diuretics unless approved | Diuretics can increase toxicity |
| Keep lab appointments | Lithium levels must be monitored |
| Report toxicity symptoms | Toxicity can become dangerous |
| Tell provider about kidney, heart, or thyroid disease | Lithium may be unsafe |
| Avoid pregnancy/breastfeeding unless provider approves | Not recommended |

---

## Lithium Monitoring

| Lab/Assessment | Why |
|---|---|
| Lithium level | Therapeutic vs toxic |
| BUN/Creatinine | Kidney function |
| Thyroid panel | Lithium can cause hypothyroidism |
| Specific gravity | Kidney concentrating ability |
| I&O | Hydration/renal effects |
| Pregnancy status if applicable | Safety concern |

---

## Lithium Adverse Effects

| Effect | Clues |
|---|---|
| Hypothyroidism | Fatigue, weight gain, cold intolerance |
| Diabetes insipidus | Excessive thirst, clear urine, large output |
| Renal impairment | Abnormal BUN/creatinine |
| Tremor | Can worsen with toxicity |
| GI upset | Nausea, vomiting, diarrhea |

---

# Anticonvulsants as Mood Stabilizers

Anticonvulsants may be used if lithium is not effective or cannot be used.

## Possible Severe Side Effects

| Side Effect |
|---|
| Liver problems |
| Coagulation problems |
| Pancreatitis |
| Bone marrow suppression |
| Dizziness |
| Headache |
| Blurred vision |
| Rash |
| Speech difficulties |
| Birth defect risk if pregnant |

<div class="warning">

Some mood stabilizers can harm a fetus. Pregnancy status and contraception teaching may be important.

</div>

# Antipsychotics

Antipsychotics are used for schizophrenia, psychosis, delusions, hallucinations, mania with psychotic features, and severe agitation.

## Types

| Type | Examples | Main Concern |
|---|---|---|
| **First-generation** | Haloperidol | Higher EPS risk |
| **Second-generation** | Risperidone, olanzapine, quetiapine | More metabolic effects |

---

## Antipsychotic Side Effects

| Side Effect | Clues | Nursing Focus |
|---|---|---|
| **EPS** | Tremor, rigidity, dystonia, restlessness | Report/treat |
| **Tardive dyskinesia** | Lip smacking, tongue movements | May be irreversible |
| **NMS** | High fever, rigidity, altered mental status, unstable VS | Emergency |
| **Sedation** | Drowsiness | Fall precautions |
| **Orthostatic hypotension** | Dizziness when standing | Rise slowly |
| **Anticholinergic effects** | Dry mouth, constipation, urinary retention | Supportive care |
| **Metabolic effects** | Weight gain, diabetes, insulin resistance | Monitor weight/glucose/lipids |

---

## EPS Quick Chart

| EPS Type | What It Looks Like |
|---|---|
| **Acute dystonia** | Sudden muscle spasms, stiff neck, eyes rolling upward |
| **Akathisia** | Restlessness, cannot sit still |
| **Parkinsonism** | Tremor, rigidity, shuffling gait |
| **Tardive dyskinesia** | Late mouth, tongue, face movements |

---

## Neuroleptic Malignant Syndrome

| Signs |
|---|
| High fever |
| Severe muscle rigidity |
| Altered mental status |
| Unstable blood pressure |
| Tachycardia |
| Sweating |

<div class="warning">

**NMS = medical emergency.** Hold medication and notify provider immediately.

</div>

# Substance-Use Medications

## Naloxone

Naloxone reverses opioid overdose.

| Key Point |
|---|
| Used for opioid overdose |
| Priority is airway/breathing first |
| Can be given intranasal, IM, SQ, or IV depending on order/protocol |
| May cause acute withdrawal |
| Monitor respirations after giving |
| Effects may wear off before opioid |

<div class="warning">

**Opioid overdose priority:** respiratory depression. Support breathing and give naloxone as ordered.

</div>

---

## Opioid Withdrawal / Maintenance

| Medication | Use |
|---|---|
| **Methadone** | Opioid maintenance/withdrawal |
| **Buprenorphine/naloxone** | Reduces cravings and withdrawal |
| **Clonidine** | Helps autonomic withdrawal symptoms |
| **Lofexidine** | Helps withdrawal symptoms |

---

## Alcohol Withdrawal Medications

| Medication | Use |
|---|---|
| **Chlordiazepoxide / Librium** | Reduces alcohol withdrawal symptoms |
| **Diazepam / Valium** | Withdrawal/seizure prevention |
| **Lorazepam / Ativan** | Withdrawal/agitation/seizure risk |
| **Thiamine / B vitamins** | Prevents/treats Wernicke-Korsakoff complications |

<div class="warning">

Alcohol withdrawal can be fatal. Monitor vital signs, hallucinations, seizures, and DTs.

</div>

---

## Disulfiram

Disulfiram (Antabuse) is alcohol aversion therapy. If alcohol is consumed, toxins build up and the patient becomes very sick.

## Alcohol Reaction

| Symptoms |
|---|
| Nausea/vomiting |
| Flushing |
| Vertigo |
| Increased pulse |
| Hypotension |
| Headache |
| Shortness of breath |
| Diaphoresis |
| Chest pain |
| Confusion |
| May be fatal |

Avoid alcohol in: drinks, cough syrup, mouthwash, sauces, aftershave, some desserts, and hidden alcohol-containing products.

---

## Tobacco / Nicotine Treatment

| Treatment |
|---|
| Nicotine replacement therapy |
| Bupropion |
| Varenicline |
| Behavioral therapy |
| Hypnosis |

# Medication Emergency Chart

| Emergency | Medication Link | Clues | Priority |
|---|---|---|---|
| **Serotonin syndrome** | SSRIs, SNRIs, MAOIs, St. John's Wort | Fever, agitation, diarrhea, tremor, hyperreflexia | Hold med, notify provider |
| **NMS** | Antipsychotics | Fever, rigidity, altered mental status, unstable VS | Medical emergency |
| **Lithium toxicity** | Lithium | GI upset, tremor, confusion, ataxia, seizures | Hold lithium, notify provider |
| **Benzo overdose** | Benzodiazepines | Sedation, respiratory depression | Airway/breathing |
| **Alcohol withdrawal/DTs** | Alcohol dependence | Tremors, high VS, hallucinations, seizures | Safety, benzos, thiamine |
| **Opioid overdose** | Opioids | Respiratory depression, decreased LOC, pinpoint pupils | Airway/breathing, naloxone |
| **Disulfiram reaction** | Disulfiram + alcohol | N/V, flushing, hypotension, chest pain | Medical assessment |

# Priority Charts

## Depression Med Priority Chart

| Situation | First Nursing Action |
|---|---|
| New antidepressant + more energy | Assess suicide risk |
| Fever, diarrhea, tremor on SSRI | Suspect serotonin syndrome |
| MAOI patient eats aged cheese and has severe headache | Assess for hypertensive crisis |
| TCA overdose concern | Cardiac/safety emergency |
| St. John's Wort with SSRI | Risk for serotonin syndrome |

---

## Bipolar Med Priority Chart

| Situation | First Nursing Action |
|---|---|
| Lithium patient has vomiting/diarrhea | Hold lithium and notify provider |
| Lithium patient has confusion/ataxia | Suspect toxicity |
| Lithium patient has excessive thirst/clear urine | Assess for diabetes insipidus |
| Patient on lithium starts diuretic | Monitor for toxicity |
| Mood stabilizer patient becomes pregnant | Notify provider |

---

## Antipsychotic Priority Chart

| Situation | First Nursing Action |
|---|---|
| Fever + rigidity | Suspect NMS; notify provider |
| Lip smacking/tongue movements | Assess tardive dyskinesia |
| Restlessness/cannot sit still | Assess akathisia |
| Sudden neck spasm/eyes rolling | Assess acute dystonia |
| Weight gain/high glucose | Monitor metabolic effects |
| Patient stops antipsychotic | Assess relapse risk |

---

## Substance Med Priority Chart

| Situation | First Nursing Action |
|---|---|
| Opioid overdose | Support breathing, give naloxone as ordered |
| Alcohol withdrawal high VS/tremors | Assess withdrawal and give ordered meds |
| DT symptoms | Medical emergency/safety |
| Disulfiram + alcohol reaction | Assess vitals and notify provider |
| Benzo withdrawal | Gradual taper/seizure precautions |

| Situation | Teaching Phrase |
|---|---|
| Antidepressant | "This medication can take several weeks to work." |
| Suicide risk | "Report any worsening mood or thoughts of self-harm." |
| Lithium | "Keep your salt and fluid intake consistent." |
| Lithium toxicity | "Report vomiting, diarrhea, confusion, tremors, or trouble walking." |
| Benzodiazepine | "Avoid alcohol and do not drive until you know how it affects you." |
| Buspirone | "This is not for immediate panic relief; it takes time to work." |

# Final Takeaway

<div class="tip">

For mental health medication questions, ask:

**1. Is this a dangerous side effect or expected side effect?** **2. Does the medication require labs?** **3. Is there overdose, withdrawal, toxicity, or respiratory depression?** **4. Does the patient need teaching about delayed effect or adherence?** **5. Should the nurse hold the medication and notify the provider?**

</div>
