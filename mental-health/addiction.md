# Addiction & Substance Use

<div class="note">

Substance use disorders are chronic conditions that affect behavior, coping, safety, physical health, relationships, and mental health.

For NCLEX, focus on: **safety, overdose, withdrawal, respiratory depression, seizures, alcohol withdrawal/DTs, nonjudgmental assessment, and relapse prevention.**

</div>

---

## Big Picture

| Concept | Meaning |
|---|---|
| **Addiction** | Chronic condition involving compulsive substance use or behavior despite harm |
| **Intoxication** | Using a substance to excess with physical/mental effects |
| **Tolerance** | Needing more substance to get the same effect |
| **Withdrawal** | Symptoms that happen when substance use stops or decreases |
| **Substance use disorder** | Disordered use that causes impairment and may lead to withdrawal |
| **Codependence** | Being overly responsible for others while ignoring own needs |
| **Dual diagnosis** | Substance use disorder plus another mental health disorder |

<div class="warning">

**NCLEX:** Overdose and withdrawal can be medical emergencies. Always think airway, breathing, circulation, seizures, and vital signs.

</div>

---

# Substance Use Assessment

Ask every patient about substance use in a calm, nonjudgmental way.

## Assessment Questions

| Question | Why It Matters |
|---|---|
| "What substances do you use?" | Identifies type of substance |
| "When was your last use?" | Predicts withdrawal timing |
| "How much do you usually use?" | Assesses severity |
| "How often do you use?" | Assesses pattern |
| "Have you tried to stop before?" | Identifies withdrawal/relapse history |
| "What happens when you stop?" | Determines withdrawal risks |
| "Do you use more than one substance?" | Polysubstance use increases risk |
| "Do you use to manage anxiety, depression, pain, or voices?" | Assesses self-medication |
| "Have you ever had seizures or DTs?" | Alcohol/benzo withdrawal danger |

<div class="note">

Patients may underestimate the amount used or the last time they used. Continue to assess objectively.

</div>

---

## General Nursing Interventions

| Intervention | Why |
|---|---|
| Provide for safety | Intoxication/withdrawal can cause injury |
| Monitor vital signs | Withdrawal often causes unstable VS |
| Monitor withdrawal symptoms | Prevent complications |
| Assess coexisting mental illness | Depression, schizophrenia, anxiety, personality disorders may coexist |
| Use nonjudgmental communication | Builds trust |
| Maintain support system | Recovery requires support |
| Explore 12-step programs | AA/NA/GA may support sobriety |
| Teach relapse prevention | Long-term management |
| Encourage therapy | Addresses coping and triggers |

# Overdose vs Withdrawal

| Term | What Happens | Nursing Priority |
|---|---|---|
| **Overdose** | Too much substance causes toxic effects | ABCs, emergency treatment |
| **Withdrawal** | Body reacts when substance is stopped/reduced | Safety, vitals, seizure precautions if needed |

<div class="note">

**Overdose** = too much substance. **Withdrawal** = body reacting to not having the substance.

</div>

# Opioids

Opioids include heroin and prescription pain medications.

## Opioid Intoxication / Overdose

Opioid overdose can cause death from **respiratory arrest**.

| Finding | Why It Matters |
|---|---|
| Respiratory depression | Priority danger |
| Decreased LOC | CNS depression |
| Pinpoint pupils | Common opioid clue |
| Hypotension | Severe CNS/respiratory depression |
| Cyanosis | Poor oxygenation |
| Snoring/gurgling respirations | Airway compromise |

---

## Opioid Overdose Nursing Priority

1. **Promote breathing**
2. **Open airway / suction if needed**
3. **Oxygen / ventilation support**
4. **Administer naloxone as ordered**
5. **Monitor for return of respiratory depression**
6. **Assess withdrawal after reversal**

<div class="warning">

**NCLEX:** For opioid overdose, breathing comes first. Naloxone is given after airway/breathing support is started.

</div>

---

## Naloxone

Naloxone is an opioid antagonist used for opioid overdose.

| Key Point |
|---|
| Reverses opioid effects |
| Can be given intranasal, IM, SQ, or IV depending on order/protocol |
| May trigger acute withdrawal |
| Patient can become agitated after reversal |
| Monitor respirations after giving |
| More doses may be needed because naloxone can wear off before opioid |

---

## Opioid Withdrawal

Opioid withdrawal is very uncomfortable but usually not as life-threatening as alcohol or benzodiazepine withdrawal.

| Symptoms |
|---|
| Anxiety |
| Muscle aches |
| Sweating |
| Runny nose |
| Dilated pupils |
| Nausea/vomiting |
| Diarrhea |
| Cramping |
| Yawning |
| Insomnia |
| Cravings |

## Opioid Withdrawal / Maintenance Meds

| Medication | Use |
|---|---|
| **Methadone** | Opioid maintenance/withdrawal management |
| **Buprenorphine/naloxone** | Reduces cravings/withdrawal |
| **Clonidine** | Helps autonomic symptoms |
| **Lofexidine** | Helps withdrawal symptoms |

# Sedatives / Hypnotics

Includes: benzodiazepines, benzodiazepine-like drugs, barbiturates, barbiturate-like hypnotics, and carbamates.

## Overdose

| Finding |
|---|
| Sedation |
| Slurred speech |
| Poor coordination |
| Respiratory depression |
| Decreased LOC |
| Hypotension |

## Withdrawal

Sedative withdrawal can be dangerous.

| Withdrawal Risk |
|---|
| Anxiety |
| Tremors |
| Insomnia |
| Agitation |
| Seizures |
| Possible life-threatening withdrawal |

<div class="warning">

**NCLEX:** Benzodiazepine/barbiturate withdrawal should be gradual to prevent seizures.

</div>

# Stimulants

Stimulants include cocaine and amphetamine-type drugs.

## Stimulant Intoxication

| Finding |
|---|
| Increased energy |
| Euphoria |
| Agitation |
| Dilated pupils |
| Tachycardia |
| Hypertension |
| Sweating |
| Decreased appetite |
| Insomnia |
| Paranoia |
| Chest pain |
| Dysrhythmias |

## Stimulant Withdrawal

| Finding |
|---|
| Fatigue |
| Depression |
| Increased sleep |
| Increased appetite |
| Cravings |
| Irritability |
| Suicidal thoughts possible |

## Nursing Care

| Intervention | Why |
|---|---|
| Monitor cardiac status | Stimulants stress heart |
| Monitor BP/HR | Hypertension/tachycardia |
| Reduce stimulation | Agitation/paranoia risk |
| Assess suicide risk during withdrawal | Depression/crash may occur |
| Therapy/support | No medication fully removes cocaine withdrawal symptoms |
| Diazepam may be ordered for agitation | Helps calm severe agitation |

# Alcohol Use Disorder

Alcohol is initially a stimulant but later has depressant effects.

Alcohol use disorder can affect: brain, liver, GI system, heart, nutrition, blood cells, hormones, mental health, and safety.

---

## Alcohol Intoxication

| Finding |
|---|
| Slurred speech |
| Poor coordination |
| Impaired judgment |
| Mood changes |
| Decreased inhibitions |
| Nausea/vomiting |
| Sedation |
| Respiratory depression at high levels |

<div class="note">

Alcohol is a CNS depressant. High levels can cause respiratory depression and death.

</div>

---

## Alcohol Withdrawal

Withdrawal symptoms may start within hours after stopping alcohol and can last for days.

| Withdrawal Finding |
|---|
| Tremors |
| Nausea/vomiting |
| Headache |
| Anorexia |
| Insomnia |
| Diarrhea |
| Anxiety |
| Sweating |
| Elevated BP |
| Pulse over 100 |
| Hallucinations |
| Seizures |

<div class="warning">

Alcohol withdrawal can be life-threatening and should be medically supervised.

</div>

---

## Alcohol Withdrawal Nursing Care

| Intervention | Why |
|---|---|
| Monitor vital signs at least q4h or per policy | Detect withdrawal worsening |
| Decrease environmental stimuli | Reduces agitation |
| Push fluids / IV fluids if ordered | Prevent dehydration |
| Seizure precautions | Withdrawal seizures may occur |
| Administer benzodiazepines as ordered | Prevents/treats severe withdrawal |
| Give B vitamins/thiamine as ordered | Prevents/treats Wernicke-Korsakoff |
| Check labs | Liver, electrolytes, nutrition, CBC concerns |
| Assess hallucinations | Safety/DT risk |
| Transfer to med-surg/ICU if severe | Medical management may be needed |

---

# Delirium Tremens

Delirium tremens, or DTs, is the most severe form of alcohol withdrawal.

It usually occurs around **72 hours** after withdrawal begins and can be fatal.

## DT Symptoms

| Finding |
|---|
| Severe agitation |
| Anxiety |
| Insomnia |
| Sweating |
| Fever possible |
| Elevated BP/HR |
| Delusions |
| Hallucinations |
| Formication |
| Seizures |
| Confusion |
| Decreased LOC possible |

<div class="note">

**Formication** = feeling like bugs are crawling on the skin.

</div>

---

## DT Treatment / Nursing Care

| Intervention | Why |
|---|---|
| Medical supervision | DTs can be fatal |
| Benzodiazepines as ordered | Controls withdrawal/agitation/seizure risk |
| IV fluids as ordered | Hydration |
| Thiamine/B vitamins | Prevents neurologic complications |
| Low-stimulation environment | Reduces agitation |
| Seizure precautions | High seizure risk |
| Monitor electrolytes/glucose | Prevent complications |
| ICU/med-surg transfer if severe | Higher level monitoring |

# Alcohol Effects on the Body

## Nutrition and Vitamin Deficiency

Alcohol use commonly causes poor nutrition and B vitamin deficiency.

| Disorder | Key Clues |
|---|---|
| **Peripheral neuropathy** | Loss of feeling in lower extremities, ulcers, paralysis risk |
| **Myopathy** | Muscle wasting/destruction |
| **Cardiac myopathy** | Enlarged heart, CHF; abstinence is essential |
| **Wernicke's encephalopathy** | Blurred vision, staggered gait, short-term memory loss |
| **Korsakoff's psychosis** | Confusion, memory loss, confabulation, gait problems, nystagmus |

<div class="warning">

Wernicke's and Korsakoff's are linked to thiamine deficiency and treated with thiamine.

</div>

---

## GI and Liver Effects

| Problem | Key Point |
|---|---|
| **Esophagitis** | Alcohol irritates GI tract |
| **Gastritis** | Stomach lining damage, ulcers, bleeding |
| **Pancreatitis** | Severe pain; pancreas digests itself |
| **Alcoholic hepatitis** | Inflamed/damaged liver |
| **Cirrhosis** | Scarred liver tissue, ascites, varices, jaundice, confusion |
| **Hepatic encephalopathy** | Ammonia/toxins build up → confusion, decreased LOC |

---

## Blood / Hormone Effects

| Effect |
|---|
| Leukopenia |
| Thrombocytopenia |
| Decreased fertility |
| Impotence in males |
| Increased bleeding risk if liver disease present |

## Disulfiram

Disulfiram (Antabuse) is aversion therapy for alcohol use disorder. It prevents alcohol from being metabolized normally, causing the patient to become very ill if alcohol is consumed.

## Reaction With Alcohol

| Symptoms |
|---|
| Nausea/vomiting |
| Flushing |
| Vertigo |
| Increased pulse |
| Hypotension |
| Headache |
| Shortness of breath |
| Diaphoresis |
| Chest pain |
| Confusion |
| May be fatal |

<div class="warning">

Patients taking disulfiram must avoid alcohol in drinks, cough syrups, mouthwash, sauces, and other hidden sources.

</div>

# Therapeutic Communication

| Patient Says | Best Response |
|---|---|
| "I can stop anytime." | "Tell me about the last time you tried to stop." |
| "I only drink to relax." | "How often do you drink to relax?" |
| "I don't have a problem." | "What concerns have others shared with you?" |
| "I used again." | "What was happening before you used?" |
| "I'm not going to AA." | "What has helped you stay sober before?" |
| "I'm ashamed." | "It sounds like this has been very difficult for you." |

Avoid: "Why can't you just stop?" / "You should know better." / "You're ruining your life." / "Just use willpower." / "You did this to yourself."

# Priority Charts

## Overdose Priority Chart

| Situation | First Nursing Action |
|---|---|
| Opioid overdose with respiratory distress | Support airway/breathing |
| Sedative overdose with decreased LOC | Airway and breathing |
| Stimulant intoxication with chest pain | Cardiac assessment/notify provider |
| Hallucinogen intoxication with panic | Quiet environment and safety |
| Alcohol intoxication with vomiting | Airway/aspiration precautions |

---

## Withdrawal Priority Chart

| Situation | First Nursing Action |
|---|---|
| Alcohol withdrawal with tremors/high VS | Assess withdrawal severity and give ordered meds |
| Alcohol withdrawal with seizure risk | Seizure precautions |
| DT symptoms | Medical emergency; notify provider/transfer if needed |
| Benzo/barbiturate withdrawal | Gradual taper/seizure precautions |
| Opioid withdrawal | Comfort meds, hydration, safety |
| Stimulant withdrawal depression | Assess suicide risk |

---

## Alcohol Priority Chart

| Situation | First Nursing Action |
|---|---|
| Pulse >100 and BP elevated after stopping alcohol | Suspect withdrawal |
| Visual/tactile hallucinations | Assess for DTs/safety |
| Confusion + gait problems + alcohol history | Think thiamine deficiency |
| Vomiting/diarrhea in withdrawal | Fluids/electrolytes |
| On disulfiram and drank alcohol | Medical emergency assessment |

# NCLEX Cues

<div class="warning">

**Opioid overdose:** respiratory depression is the priority.

</div>

<div class="warning">

**Naloxone reverses opioids** but monitor respirations because effects can wear off.

</div>

<div class="warning">

**Alcohol withdrawal can be fatal.** Monitor VS, seizure risk, hallucinations, and DTs.

</div>

<div class="warning">

**DTs usually occur around 72 hours after alcohol withdrawal begins.**

</div>

<div class="note">

**Alcohol withdrawal treatment commonly uses benzodiazepines** such as chlordiazepoxide, diazepam, or lorazepam.

</div>

<div class="note">

**Thiamine is important in alcohol use disorder** to prevent/treat Wernicke-Korsakoff complications.

</div>

<div class="warning">

**Benzo/barbiturate withdrawal requires gradual reduction** to prevent seizures.

</div>

<div class="note">

**Stimulant withdrawal can cause depression.** Assess suicide risk.

</div>

<div class="warning">

**Disulfiram + alcohol can be dangerous or fatal.**

</div>

<div class="note">

**Use nonjudgmental questions.** Shame decreases honesty and trust.

</div>

# Final Takeaway

<div class="tip">

For addiction and substance use questions, ask:

**1. Is this overdose or withdrawal?** **2. Is airway or breathing at risk?** **3. Is there seizure risk?** **4. Are vital signs unstable?** **5. Does the patient need withdrawal medication or emergency reversal?** **6. Am I using nonjudgmental communication and supporting recovery?**

</div>
