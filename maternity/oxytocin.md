# Oxytocin

## Medication

|                 |            |
| --------------- | ---------- |
| Medication Name | Oxytocin   |
| Brand Name      | Pitocin    |
| Category/Class  | Uterotonic |

## Purpose / Expected Action

* Stimulates uterine contractions.
* Strengthens or starts labor contractions.
* Helps the uterus contract after birth to decrease bleeding.

## Therapeutic Use

* Labor induction
* Labor augmentation
* Postpartum hemorrhage prevention/treatment
* Uterine atony after birth

## Route

* IV infusion for labor induction or augmentation
* IV or IM postpartum depending on order/facility policy

## Complications / Side Effects

* Uterine tachysystole
* Nonreassuring fetal heart rate
* Fetal distress
* Uterine rupture
* Hypotension
* Water intoxication with high doses or prolonged use

## Contraindications / Precautions

Use caution or avoid with:

* Nonreassuring fetal heart rate pattern
* Uterine tachysystole
* Placenta previa
* Abnormal fetal position
* Previous classical uterine incision
* Any condition where vaginal birth is contraindicated

## Nursing Interventions

* Administer using an IV pump.
* Monitor fetal heart rate continuously during induction or augmentation.
* Monitor contraction frequency, duration, intensity, and resting tone.
* Assess maternal vital signs.
* Assess for uterine tachysystole.
* Monitor intake and output if receiving high-dose or prolonged oxytocin.
* Keep patient on continuous assessment because fetal distress can happen quickly.

## If Tachysystole or Fetal Distress Occurs

1. Stop oxytocin.
2. Turn patient to side-lying position.
3. Increase IV fluids.
4. Notify provider.
5. Prepare to give a tocolytic if ordered.

## Client Education

* This medication helps start or strengthen contractions.
* Contractions and fetal heart rate will be monitored closely.
* Report severe pain, constant pressure, dizziness, shortness of breath, or if contractions do not seem to relax.

## Evaluation of Medication Effectiveness

Medication is effective if:

* Contractions become adequate.
* Cervix changes during labor.
* Fetal heart rate remains reassuring.
* Postpartum uterus becomes firm.
* Postpartum bleeding decreases.

## NCLEX / ATI Must Know

If oxytocin causes too many contractions or fetal distress, the priority action is to **stop the oxytocin**.

## Memory Hook

**Pitocin = Pushes contractions.**
