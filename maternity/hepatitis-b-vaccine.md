# Hepatitis B Vaccine

## Medication

|                 |                     |
| --------------- | ------------------- |
| Medication Name | Hepatitis B Vaccine |
| Category/Class  | Vaccine             |

## Purpose / Expected Action

* Stimulates the newborn’s immune system to make antibodies against hepatitis B.
* Provides active immunity.
* Helps prevent hepatitis B infection.

## Therapeutic Use

* Routine newborn immunization.
* Prevention of hepatitis B transmission.

## Route / Site

* IM injection
* Given in the vastus lateralis

## Timing

* Given after birth, usually within 12 hours.
* If the mother is hepatitis B positive or status is unknown, the newborn needs protection as soon as possible.

## If Mother Is Hepatitis B Positive

Newborn should receive:

* Hepatitis B vaccine
* HBIG

These are given:

* Within 12 hours of birth
* At separate injection sites

## Complications / Side Effects

* Mild fever
* Soreness at injection site
* Redness or swelling at injection site
* Rare allergic reaction

## Contraindications / Precautions

Use caution with:

* Severe allergy to vaccine or vaccine components

## Nursing Interventions

* Check maternal hepatitis B status.
* Verify consent/facility policy.
* Give IM in the vastus lateralis.
* If mother is hepatitis B positive, make sure HBIG is also given.
* Give vaccine and HBIG in separate injection sites if both are ordered.
* Monitor injection site.
* Document time, site, and newborn response.

## Client Education

* The vaccine helps the baby build protection against hepatitis B.
* It is part of routine newborn care.
* If the mother has hepatitis B, the baby also needs HBIG for immediate protection.

## Evaluation of Medication Effectiveness

Medication is effective if:

* Vaccine is administered correctly.
* No serious reaction occurs.
* Follow-up vaccine schedule is continued.

## NCLEX / ATI Must Know

Hepatitis B vaccine gives **active immunity**.

HBIG gives **passive immunity**.

If the mother is hepatitis B positive, the newborn receives **both Hepatitis B vaccine and HBIG**.

## Memory Hook

**Hep B vaccine = active protection. HBIG = immediate passive protection.**
