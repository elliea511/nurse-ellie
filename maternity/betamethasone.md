# Betamethasone

## Medication

|                 |                |
| --------------- | -------------- |
| Medication Name | Betamethasone  |
| Category/Class  | Corticosteroid |

## Purpose / Expected Action

* Stimulates fetal surfactant production.
* Helps mature the fetal lungs before preterm birth.
* Decreases risk of newborn respiratory distress.

## Therapeutic Use

* Given when preterm birth is expected.
* Promotes fetal lung maturity.

## Route

* IM injection

## Given When

* Commonly given between 24-34 weeks gestation when preterm birth is likely.
* May be given before anticipated preterm delivery if ordered.

## Complications / Side Effects

* Increased maternal blood glucose
* Temporary increase in WBC count
* Possible temporary decrease in fetal movement
* Possible temporary changes in fetal heart rate variability

## Contraindications / Precautions

Use caution with:

* Diabetes or gestational diabetes
* Active infection
* Maternal hyperglycemia

## Nursing Interventions

* Assess gestational age.
* Assess contractions and signs of preterm labor.
* Monitor maternal blood glucose, especially if diabetic.
* Monitor fetal status as ordered.
* Monitor for signs of infection.
* Teach that this medication is for fetal lung maturity.

## Client Education

* This medication helps the baby’s lungs mature.
* It does not stop contractions.
* Blood sugar may increase temporarily.
* Report fever, worsening symptoms, or signs of infection.

## Evaluation of Medication Effectiveness

Medication is effective if:

* Medication course is completed before preterm birth when possible.
* Newborn has improved respiratory outcomes.
* Maternal blood glucose remains monitored and managed.

## NCLEX / ATI Must Know

Betamethasone is given for **fetal lung maturity**, not to stop labor.

It can increase maternal blood glucose.

## Memory Hook

**Betamethasone = Baby breathing.**
