# Hemabate

## Medication

|                 |                          |
| --------------- | ------------------------ |
| Medication Name | Hemabate                 |
| Generic Name    | Carboprost tromethamine  |
| Category/Class  | Prostaglandin uterotonic |

## Purpose / Expected Action

* Causes strong uterine contractions.
* Helps firm the uterus after birth.
* Decreases postpartum bleeding caused by uterine atony.

## Therapeutic Use

* Postpartum hemorrhage
* Uterine atony that does not respond well enough to other interventions

## Route

* IM injection

## Complications / Side Effects

* Diarrhea
* Nausea/vomiting
* Fever
* Chills
* Abdominal cramping
* Hypertension
* Bronchospasm

## Contraindications / Precautions

Do not give with:

* Asthma

Use caution with:

* Cardiac disease
* Renal disease
* Hepatic disease
* Hypertension

## Nursing Interventions

* Assess for asthma before giving.
* Assess respiratory status.
* Monitor uterine tone.
* Monitor lochia and amount of bleeding.
* Monitor vital signs.
* Monitor for severe diarrhea, nausea, vomiting, or fever.
* Notify provider if bronchospasm or respiratory distress occurs.

## Client Education

* Cramping is expected because the medication makes the uterus contract.
* Diarrhea, nausea, or vomiting may occur.
* Report shortness of breath, wheezing, chest tightness, or heavy bleeding.

## Evaluation of Medication Effectiveness

Medication is effective if:

* Fundus becomes firm.
* Bleeding decreases.
* Vital signs remain stable.
* No signs of continued hemorrhage occur.

## NCLEX / ATI Must Know

Hemabate is used for postpartum hemorrhage, but it is **held if the patient has asthma** because it can cause bronchospasm.

## Memory Hook

**Hemabate = Helps hemorrhage, but harms asthma.**
