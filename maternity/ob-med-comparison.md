# OB Medication Exam Comparison

## Quick Comparison

| Medication          | Main Use                                              | Biggest Exam Warning                                           |
| ------------------- | ----------------------------------------------------- | -------------------------------------------------------------- |
| Methergine          | Postpartum hemorrhage / uterine atony                 | Do not give with hypertension or preeclampsia                  |
| Hemabate            | Postpartum hemorrhage / uterine atony                 | Do not give with asthma                                        |
| Oxytocin            | Induction, augmentation, postpartum hemorrhage        | Stop for tachysystole or fetal distress                        |
| Magnesium Sulfate   | Preeclampsia seizure prevention / eclampsia treatment | Toxicity = absent reflexes, low respirations, low urine output |
| Betamethasone       | Fetal lung maturity                                   | Can increase maternal blood glucose                            |
| RhoGAM              | Prevents Rh sensitization                             | Give to Rh-negative mother, not baby                           |
| Vitamin K           | Prevents newborn bleeding                             | Helps clotting factors II, VII, IX, and X                      |
| Hepatitis B Vaccine | Prevents Hepatitis B infection                        | If mother is Hep B positive, newborn also needs HBIG           |

# Exam Triggers

| If the Question Says...                           | Think...                           |
| ------------------------------------------------- | ---------------------------------- |
| Postpartum hemorrhage + high blood pressure       | Hold Methergine                    |
| Postpartum hemorrhage + preeclampsia              | Hold Methergine                    |
| Postpartum hemorrhage + asthma                    | Hold Hemabate                      |
| Too many contractions on Pitocin                  | Stop oxytocin                      |
| Nonreassuring fetal heart rate on Pitocin         | Stop oxytocin                      |
| Preeclampsia + absent reflexes                    | Magnesium toxicity                 |
| Magnesium toxicity antidote                       | Calcium gluconate                  |
| Respirations less than 12 on magnesium            | Hold magnesium and notify provider |
| Urine output less than 30 mL/hr on magnesium      | Hold magnesium and notify provider |
| Preterm labor + fetal lungs                       | Betamethasone                      |
| Rh-negative mother at 28 weeks                    | RhoGAM                             |
| Rh-negative mother + Rh-positive baby after birth | RhoGAM within 72 hours             |
| Newborn bleeding prevention                       | Vitamin K                          |
| Newborn Hep B prevention                          | Hepatitis B vaccine                |
| Hep B positive mother                             | Hepatitis B vaccine + HBIG         |

# Postpartum Hemorrhage Medications

| Medication | What It Does                                 | Do Not Give If                             |
| ---------- | -------------------------------------------- | ------------------------------------------ |
| Oxytocin   | Stimulates uterine contractions              | Tachysystole / fetal distress during labor |
| Methergine | Causes strong sustained uterine contractions | Hypertension / preeclampsia                |
| Hemabate   | Causes uterine contractions                  | Asthma                                     |

## Easy Way to Remember

* **Methergine = BP**
* **Hemabate = Asthma**
* **Oxytocin = Contractions/FHR**

# Priority Actions

## Oxytocin Problem

If patient has tachysystole or fetal distress:

1. Stop oxytocin.
2. Turn patient side-lying.
3. Increase IV fluids.
4. Notify provider.
5. Prepare for tocolytic if ordered.

## Magnesium Toxicity

If patient has absent reflexes, low respirations, or low urine output:

1. Stop magnesium sulfate.
2. Notify provider.
3. Prepare calcium gluconate.
4. Support respirations.
5. Continue maternal and fetal monitoring.

## Postpartum Hemorrhage

If fundus is boggy:

1. Massage fundus.
2. Assess bleeding.
3. Empty bladder.
4. Give uterotonic as ordered.
5. Continue monitoring vital signs and lochia.

# Must-Know One-Liners

* **Methergine:** Do not give with hypertension.
* **Hemabate:** Do not give with asthma.
* **Oxytocin:** Stop first if tachysystole or fetal distress occurs.
* **Magnesium Sulfate:** Monitor respirations, reflexes, and urine output.
* **Betamethasone:** Helps fetal lungs mature.
* **RhoGAM:** Given to Rh-negative mother.
* **Vitamin K:** Prevents newborn bleeding.
* **Hepatitis B Vaccine:** Active immunity against Hep B.
* **HBIG:** Passive immunity if mother is Hep B positive.
