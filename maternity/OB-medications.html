postpartum hemorrhage | | Magnesium Sulfate | Preeclampsia seizure prevention; neuroprotection | | Hemabate | Postpartum hemorrhage | | Hepatitis B Vaccine | Newborn immunization | | Betamethasone | Fetal lung maturity | | RhoGAM | Prevent Rh sensitization | --- # Methergine ## Generic Name Methylergonovine ## Drug Class Uterotonic / ergot alkaloid ## Used For * Postpartum hemorrhage * Uterine atony * Helps uterus contract firmly after birth ## Priority Assessment | Assess | Why | | --------------- | ------------------------------------ | | Blood pressure | Can cause severe hypertension | | Uterine tone | Should become firm | | Bleeding/lochia | Used to decrease bleeding | | Pain/cramping | Strong uterine contractions expected | ## Do NOT Give If * Hypertension * Preeclampsia * Eclampsia * Severe cardiac disease ## Side Effects * Hypertension * Nausea/vomiting * Cramping * Headache * Chest pain, if severe complication ## NCLEX Must Know Methergine is held if the patient has **high blood pressure**. --- # Vitamin K ## Generic Name Phytonadione ## Drug Class Vitamin / bleeding prevention medication ## Used For * Prevents hemorrhagic disease of the newborn * Helps newborn form clotting factors ## Why Newborns Need It Newborns have low vitamin K because: * Sterile gut cannot make enough vitamin K yet * Liver is immature * Breast milk has low vitamin K ## Route * IM injection * Usually given in the vastus lateralis ## Nursing Assessment | Assess | Why | | -------------------- | ------------------------------------ | | Injection site | Monitor for bleeding or irritation | | Signs of bleeding | Vitamin K prevents abnormal bleeding | | Parent understanding | Parents may ask why baby needs it | ## NCLEX Must Know Vitamin K helps produce clotting factors **II, VII, IX, and X**. Refusal increases newborn risk for bleeding. --- # Oxytocin ## Brand Name Pitocin ## Drug Class Uterotonic ## Used For * Induction of labor * Augmentation of labor * Postpartum hemorrhage prevention/treatment * Helps uterus contract ## Priority Assessment | Assess | Why | | ------------------------------------------ | -------------------------------------- | | Contraction frequency, duration, intensity | Can cause tachysystole | | Fetal heart rate | Fetal distress can occur | | Maternal vital signs | Monitor for complications | | Uterine resting tone | Uterus must relax between contractions | | Bleeding postpartum | Should decrease with firm uterus | ## Side Effects / Complications * Uterine tachysystole * Fetal distress * Uterine rupture * Water intoxication with prolonged/high-dose use * Hypotension ## If Tachysystole or Fetal Distress Occurs 1. Stop oxytocin. 2. Turn patient to side. 3. Increase IV fluids. 4. Notify provider. 5. Prepare to give tocolytic if ordered. ## NCLEX Must Know Oxytocin requires **continuous fetal monitoring** when used for induction or augmentation. --- # Magnesium Sulfate ## Drug Class Anticonvulsant / CNS depressant ## Used For * Prevents seizures in preeclampsia * Treats eclampsia * May be used for fetal neuroprotection in preterm labor ## Priority Assessment | Assess | Why | | ---------------------- | ------------------------------------------ | | Respiratory rate | Magnesium can cause respiratory depression | | Deep tendon reflexes | Loss of reflexes = toxicity | | Urine output | Excreted by kidneys | | Level of consciousness | CNS depression can occur | | Blood pressure | Often used with severe preeclampsia | | Fetal status | Monitor baby during therapy | ## Therapeutic Effects * Prevents seizures * Decreases neuromuscular irritability ## Signs of Toxicity * Respiratory rate less than 12/min * Absent or decreased deep tendon reflexes * Urine output less than 30 mL/hr * Decreased LOC * Cardiac dysrhythmias * Respiratory arrest ## Antidote Calcium gluconate ## NCLEX Must Know Before giving magnesium sulfate, check: * Respirations * Reflexes * Urine output Hold and notify provider if respirations are low, reflexes are absent, or urine output is poor. --- # Hemabate ## Generic Name Carboprost tromethamine ## Drug Class Prostaglandin uterotonic ## Used For * Postpartum hemorrhage * Uterine atony ## Priority Assessment | Assess | Why | | ------------------- | ----------------------------------- | | Respiratory history | Contraindicated/caution with asthma | | Uterine tone | Should become firm | | Bleeding | Used to decrease hemorrhage | | GI symptoms | Causes diarrhea, nausea, vomiting | | Temperature | Can cause fever | ## Do NOT Give If * Asthma * Use caution with cardiac, renal, or hepatic disease ## Side Effects * Diarrhea * Nausea/vomiting * Fever * Bronchospasm * Hypertension * Uterine cramping ## NCLEX Must Know Hemabate is avoided in patients with **asthma** because it can cause bronchospasm. --- # Hepatitis B Vaccine ## Drug Class Vaccine ## Used For * Prevents hepatitis B infection in newborn ## Timing * Given after birth * Usually within 12 hours for newborns, especially if mother is Hep B positive or status unknown ## Route * IM injection * Vastus lateralis ## Nursing Assessment | Assess | Why | | --------------------- | ------------------------------------- | | Maternal Hep B status | Determines if newborn also needs HBIG | | Consent/policy | Follow facility protocol | | Injection site | Monitor for redness/swelling | | Allergies | Check vaccine contraindications | ## If Mother Is Hep B Positive Newborn needs: * Hepatitis B vaccine * HBIG Both should be given within 12 hours of birth, at separate injection sites. ## NCLEX Must Know Hep B vaccine gives **active immunity**. HBIG gives **passive immunity**. --- # Betamethasone ## Drug Class Corticosteroid ## Used For * Promotes fetal lung maturity * Given when preterm birth is expected ## Main Benefit Stimulates surfactant production, which decreases risk of: * Respiratory distress syndrome * Intraventricular hemorrhage * Necrotizing enterocolitis ## Given To Pregnant patient at risk for preterm delivery, usually between **24-34 weeks** gestation. ## Nursing Assessment | Assess | Why | | ------------------------ | ------------------------------------- | | Gestational age | Used for fetal lung maturity | | Blood glucose | Steroids can increase glucose | | Contractions/labor signs | Given when preterm birth risk exists | | Infection signs | Steroids can suppress immune response | ## Side Effects * Increased maternal blood glucose * Possible temporary changes in fetal movement/FHR variability * Increased WBC count ## NCLEX Must Know Betamethasone is for the **baby’s lungs**, not to stop contractions. --- # RhoGAM ## Generic Name Rho(D) immune globulin ## Drug Class Immune globulin ## Used For Prevents Rh sensitization in an Rh-negative mother. ## Given When Mother is: * Rh-negative And baby/fetus is or may be: * Rh-positive ## Common Times Given * Around 28 weeks gestation * Within 72 hours after birth if newborn is Rh-positive * After miscarriage * After abortion * After ectopic pregnancy * After amniocentesis * After abdominal trauma * After bleeding during pregnancy ## Priority Assessment | Assess | Why | | ------------------------------------- | -------------------------------- | | Maternal blood type | Must be Rh-negative | | Baby blood type after birth | Determines postpartum need | | Antibody screen | Should not already be sensitized | | History of bleeding/trauma/procedures | May require dose | ## NCLEX Must Know RhoGAM is given to the **mother**, not the baby. It prevents future Rh problems. It does **not** treat the current baby’s hemolytic disease. --- # Quick OB Med Comparison | Medication | Use | Biggest Hold/Concern | | ----------------- | --------------------------------- | ---------------------------------------- | | Methergine | Postpartum hemorrhage | Hypertension/preeclampsia | | Hemabate | Postpartum hemorrhage | Asthma | | Oxytocin | Labor induction/augmentation, PPH | Tachysystole/fetal distress | | Magnesium sulfate | Preeclampsia seizures | Low RR, absent reflexes, low UOP | | Betamethasone | Fetal lung maturity | Hyperglycemia | | RhoGAM | Prevent Rh sensitization | Give only to Rh-negative mother | | Vitamin K | Prevent newborn bleeding | Refusal increases bleeding risk | | Hep B vaccine | Prevent Hep B | If mother positive, baby also needs HBIG | --- # Must-Know Exam Triggers | Exam Clue | Think | | -------------------------------- | ---------------------- | | PPH + high BP | Do not give Methergine | | PPH + asthma | Do not give Hemabate | | Too many contractions on Pitocin | Stop oxytocin first | | Preeclampsia + absent reflexes | Magnesium toxicity | | Magnesium antidote | Calcium gluconate | | Preterm labor + fetal lungs | Betamethasone | | Rh-negative mother | RhoGAM | | Newborn clotting | Vitamin K | | Newborn Hep B prevention | Hep B vaccine | | Hep B positive mother | Hep B vaccine + HBIG | ``` Your uploaded newborn notes also list Vitamin K, Hepatitis B vaccine, and newborn medication/screening points. :contentReference[oaicite:0]{index=0} ```
