# ATI Nutrition Exam Study Guide: Condensed Review

<h2 class="section-divider" id="nutrition-basics-condensed">Nutrition Basics / Foundations</h2>

## ATI Nutrition Big Picture

ATI nutrition questions usually test:

| Question Type                   | What to Think                                                    |
| ------------------------------- | ---------------------------------------------------------------- |
| **Best food choice**            | Which food fits the client’s condition?                          |
| **Deficiency clue**             | What missing nutrient causes this symptom?                       |
| **Priority finding**            | Is the client dehydrated, malnourished, overloaded, or unstable? |
| **Diet teaching**               | What is realistic, safe, and disease-specific?                   |
| **Medication-food interaction** | Does this nutrient/food affect the medication?                   |

<div class="nclex">
Do not just memorize foods. Ask: <strong>What does this nutrient do, and why does this client need or avoid it?</strong>
</div>

---

## RDA / DRI

| Term    | Meaning                                                                             |
| ------- | ----------------------------------------------------------------------------------- |
| **RDA** | Recommended Dietary Allowance; daily intake that meets needs of most healthy people |
| **DRI** | Dietary Reference Intake; broad category of nutrient reference values               |
| **AI**  | Adequate Intake; used when RDA is not available                                     |
| **UL**  | Tolerable Upper Intake Level; highest daily amount unlikely to cause harm           |

<div class="note">
RDA values are for generally healthy people. Illness, pregnancy, wounds, burns, infection, and malnutrition can change needs.
</div>

---

## Macronutrients, Digestion, and Enzymes

| Macronutrient                | Main Function                                                   | Enzyme        | Breaks Down Into      | ATI Clue                                          |
| ---------------------------- | --------------------------------------------------------------- | ------------- | --------------------- | ------------------------------------------------- |
| **Carbohydrates / starches** | Preferred energy source                                         | **Amylase**   | Glucose/simple sugars | Diabetes, dumping syndrome, energy                |
| **Protein**                  | Tissue repair, muscle, immunity, fluid balance                  | **Proteases** | Amino acids           | Wounds, infection, surgery, burns, malnutrition   |
| **Fat/lipids**               | Stored energy, insulation, organ protection, vitamin absorption | **Lipase**    | Fatty acids/glycerol  | Fat-soluble vitamins, pancreatitis, bile problems |
| **Lactose**                  | Dairy sugar                                                     | **Lactase**   | Glucose/galactose     | Lactose intolerance                               |

### Energy use order

1. **Glucose from carbohydrates** = preferred/first energy source
2. **Protein** = used if carbs/calories are inadequate; should be spared for healing
3. **Fat** = stored/concentrated energy

<div class="danger">
If calorie or carbohydrate intake is too low, the body can break down <strong>protein</strong> for energy, worsening muscle wasting, weakness, poor healing, and malnutrition.
</div>

### Quick enzyme clues

| Finding / Condition                    | Think                         |
| -------------------------------------- | ----------------------------- |
| Starches/carbohydrates                 | Amylase                       |
| Protein                                | Protease                      |
| Fat/lipids                             | Lipase                        |
| Gas/diarrhea after dairy               | Lactase deficiency            |
| Greasy, floating, foul-smelling stools | Fat malabsorption             |
| Clay-colored stool + jaundice          | Bile flow problem             |
| Pancreatitis/cystic fibrosis           | Pancreatic enzyme impairment  |
| Celiac disease                         | Damaged villi → malabsorption |

## Micronutrients

| Nutrient       | Main Function                                        | ATI Clue                                                      |
| -------------- | ---------------------------------------------------- | ------------------------------------------------------------- |
| **Vitamin A**  | Vision, skin, mucous membranes, immunity             | Night vision                                                  |
| **Lutein**     | Eye health                                           | Kale, spinach, collards, mustard greens; macular degeneration |
| **Vitamin B6** | Heme formation, neurotransmitters                    | Blood + nervous system                                        |
| **Folate**     | RBC formation, neural tube development               | Pregnancy; chicken liver high in folate                       |
| **Vitamin C**  | Collagen, wound healing, iron absorption             | Give with iron; avoid citrus with mucositis                   |
| **Vitamin D**  | Calcium absorption, bone health                      | Older adults; bone loss                                       |
| **Vitamin K**  | Clotting                                             | Warfarin = consistent intake                                  |
| **Calcium**    | Bones, teeth, nerve transmission, muscle contraction | Calcium carbonate with meals                                  |
| **Magnesium**  | Nerve, muscle, cardiac function                      | Can be low with TPN                                           |
| **Potassium**  | Cardiac rhythm, muscle contraction                   | ACE inhibitors can cause hyperkalemia                         |
| **Phosphorus** | Bones/teeth, hormone regulation                      | Not night vision                                              |
| **Iron**       | Hemoglobin/oxygen transport                          | Pregnancy: 30 mg/day; take with vitamin C                     |

### Fat-soluble vitamins

| Type              | Vitamins     | Main Point                           |
| ----------------- | ------------ | ------------------------------------ |
| **Fat-soluble**   | A, D, E, K   | Stored in body; toxicity risk higher |
| **Water-soluble** | B-complex, C | Excess usually excreted in urine     |

## Malnutrition

| Finding                      | Possible Meaning                     |
| ---------------------------- | ------------------------------------ |
| Lower extremity edema        | Protein deficiency                   |
| Dry skin/conjunctiva         | Poor nutrition or vitamin deficiency |
| Corneal vascularization      | Malnutrition                         |
| Paresthesia/weak hand grasps | Vitamin deficiency or muscle wasting |
| Poor wound healing           | Low protein or vitamin C             |
| Weight loss                  | Poor intake, illness, malabsorption  |

| Lab            | ATI Meaning                                                  |
| -------------- | ------------------------------------------------------------ |
| **Albumin**    | Longer-term protein status; affected by illness/inflammation |
| **Prealbumin** | Better for recent/severe malnutrition                        |

<div class="danger">
Poor intake + weight loss + edema = possible <strong>protein deficiency/malnutrition</strong>.
</div>

## Braden Scale: Nutrition Scoring

| Score | Category            | ATI Clue                                                               |
| ----- | ------------------- | ---------------------------------------------------------------------- |
| **1** | Very Poor           | Eats only bites; rarely eats a meal; NPO/clear liquids without support |
| **2** | Probably Inadequate | Eats about half; limited protein; supplements only sometimes           |
| **3** | Adequate            | Eats more than half of most meals; occasional refusal; enough protein  |
| **4** | Excellent           | Eats almost all meals; rarely refuses; protein intake strong           |

<div class="nclex">
Client eats more than half of most meals, occasionally refuses, and has four protein servings/day = <strong>3, Adequate</strong>.
</div>

## Fluid Balance

| Fluid Deficit               | Fluid Excess        |
| --------------------------- | ------------------- |
| Vomiting/diarrhea           | Edema               |
| Dry mucous membranes        | Crackles            |
| Poor skin turgor            | Shortness of breath |
| Tachycardia                 | Hypertension        |
| Orthostatic hypotension     | Bounding pulses     |
| Dark urine/decreased output | Rapid weight gain   |

<div class="danger">
Vomiting and diarrhea increase risk for <strong>fluid volume deficit</strong> and electrolyte imbalance.
</div>

## Healthy Weight Loss

| Teaching                    | ATI Point                       |
| --------------------------- | ------------------------------- |
| Lose **0.5–1 lb/week**      | Safe, realistic goal            |
| Use shopping list           | Prevents impulse buying         |
| Eat low-calorie foods first | Helps fullness                  |
| Avoid tasting while cooking | Prevents hidden calories        |
| Portion control             | Better than extreme restriction |
| Regular meals               | Prevents overeating             |

<h2 class="section-divider" id="gi-condensed">GI Disorders and Diet Therapy</h2>

## GI Disorders: High-Yield

| Disorder                | What Happens                                | Diet Teaching                                                   |
| ----------------------- | ------------------------------------------- | --------------------------------------------------------------- |
| **Celiac disease**      | Gluten damages villi → malabsorption        | Avoid wheat, barley, rye, malt                                  |
| **Gluten-free foods**   | Safe choices                                | Tapioca, potato flour, beans, corn, quinoa, plain rice          |
| **Dumping syndrome**    | Food leaves stomach too fast → fluid shifts | Small frequent meals, fluids between meals, avoid simple sugars |
| **Gastric bypass**      | Small pouch, poor intake risk               | Protein first; 60–120 g/day protein                             |
| **Pancreatitis**        | Inflamed pancreas; insulin may decrease     | Watch glucose, hydration, electrolytes                          |
| **Cirrhosis/ascites**   | Liver dysfunction + fluid retention         | Sodium restriction, daily weight, fluid monitoring              |
| **Diverticulosis**      | Pouches, no inflammation                    | High fiber + fluids                                             |
| **Diverticulitis**      | Inflamed/infected pouches                   | Low fiber/low residue during acute flare if prescribed          |
| **IBD/Crohn’s**         | Inflammation, diarrhea, malabsorption       | Low residue during flares; monitor weight/deficiencies          |
| **IBS**                 | Functional bowel symptoms                   | Peppermint oil/probiotics may help; avoid triggers              |
| **Lactose intolerance** | Low lactase                                 | Lactose-free products; maintain calcium/vitamin D               |
| **Ostomy**              | Stool/output management                     | Pectin/starches thicken stool                                   |
| **Ileostomy**           | Liquid output/electrolyte loss              | Increase fluids, salt if prescribed, monitor B12                |
| **Constipation**        | Low fiber/fluid/activity                    | Increase fiber gradually + fluids                               |
| **Diarrhea**            | Fluid/electrolyte loss                      | Hydration and electrolytes are priority                         |

### Dumping syndrome clues

Avoid:

* Candy
* Syrup
* Sweet drinks
* Sugar alcohols
* High-sugar desserts

Teach:

* Small frequent meals
* Fluids between meals
* Protein with meals
* Complex carbs
* Lie down after meals if instructed

<h2 class="section-divider" id="diabetes-condensed">Endocrine and Diabetes Nutrition</h2>

## Diabetes Basics

| Topic                | ATI Point                                              |
| -------------------- | ------------------------------------------------------ |
| **Type 1 diabetes**  | Little/no insulin; insulin required; high DKA risk     |
| **Type 2 diabetes**  | Insulin resistance; nutrition/activity improve control |
| **Carbohydrates**    | Most direct effect on glucose                          |
| **Consistent carbs** | Prevents glucose swings                                |
| **Glycemic index**   | Baked potato = high glycemic index                     |
| **HbA1c**            | Average glucose over 2–3 months                        |
| **A1c 8.7%**         | Poor control/not expected goal                         |

## Hypoglycemia vs Hyperglycemia

| Hypoglycemia     | Hyperglycemia      |
| ---------------- | ------------------ |
| Cool, clammy     | Dry skin           |
| Shaky            | Thirsty            |
| Sweaty           | Polyuria           |
| Confused/erratic | Fatigue            |
| Hungry           | Blurred vision     |
| Tachycardia      | Slow wound healing |

<div class="nclex">
If symptoms occur, check <strong>blood glucose</strong> first when possible.
</div>

## DKA vs HHS

| Feature    | DKA                                  | HHS                                       |
| ---------- | ------------------------------------ | ----------------------------------------- |
| Usually    | Type 1                               | Type 2                                    |
| Glucose    | High                                 | Very high                                 |
| Ketones    | Present                              | Minimal/absent                            |
| Acidosis   | Yes                                  | Usually no significant acidosis           |
| Main issue | Insulin deficiency + ketones         | Severe dehydration/hyperosmolar state     |
| Clues      | Fruity breath, Kussmaul respirations | Altered mental status, severe dehydration |

## Morning Hyperglycemia

| Topic                  | Meaning                                              |
| ---------------------- | ---------------------------------------------------- |
| **Somogyi phenomenon** | Morning hyperglycemia after nighttime hypoglycemia   |
| **Dawn phenomenon**    | Morning hyperglycemia without nighttime hypoglycemia |

<h2 class="section-divider" id="cardio-renal-condensed">Cardiovascular and Renal Nutrition</h2>

## Cardiovascular Nutrition

| Topic                    | ATI Teaching                                           |
| ------------------------ | ------------------------------------------------------ |
| **Hypertension**         | Sodium around 1,500 mg/day can help lower BP           |
| **Salt substitutes**     | May contain potassium; caution with CKD/ACE inhibitors |
| **Plant proteins**       | Can help lower cholesterol                             |
| **Saturated/trans fats** | Limit to reduce CVD risk                               |
| **Niacin**               | Lowers LDL and triglycerides                           |
| **Multivitamins**        | Do not prevent CVD if diet is balanced                 |
| **Exercise**             | 150 min moderate activity/week for prevention          |

## Renal Nutrition

| Topic             | ATI Point                                |
| ----------------- | ---------------------------------------- |
| **CKD**           | Diet depends on labs and dialysis status |
| **Potassium**     | Restrict if high; watch salt substitutes |
| **Phosphorus**    | May need restriction if high             |
| **Sodium**        | Restrict for HTN/fluid retention         |
| **Fluid**         | May need restriction if fluid overloaded |
| **Protein**       | Depends on CKD stage and dialysis        |
| **Kidney stones** | Increase fluids if not contraindicated   |

<h2 class="section-divider" id="resp-neuro-condensed">Respiratory and Neurologic Nutrition</h2>

## COPD Nutrition

| Problem                     | Teaching                                                  |
| --------------------------- | --------------------------------------------------------- |
| Fatigue with meals          | Small frequent meals                                      |
| Weight loss                 | High-calorie, high-protein foods                          |
| Increased work of breathing | Rest before meals                                         |
| Dyspnea while eating        | Avoid gas-producing foods if they worsen symptoms         |
| Need for pulmonary formula  | High-calorie, high-protein, low/moderate carb, higher fat |

<div class="note">
COPD clients may need more calories because breathing takes extra energy.
</div>

## Parkinson’s Disease

| Issue             | Nutrition Focus              |
| ----------------- | ---------------------------- |
| Rigidity/tremors  | Increased calorie needs      |
| Dysphagia risk    | Swallow precautions          |
| Constipation      | Fiber/fluids if appropriate  |
| Medication timing | Follow provider instructions |

## Aspiration Precautions

* Sit upright at 90 degrees
* Chin tuck/head forward if recommended
* Place food on stronger/unaffected side
* Clear mouth before liquids
* Use thickened liquids if prescribed
* Assess swallow before oral intake after stroke

<h2 class="section-divider" id="cancer-condensed">Cancer, Immunity, and Wound Healing Nutrition</h2>

## Cancer Prevention

| Teaching                           | ATI Point                     |
| ---------------------------------- | ----------------------------- |
| Maintain healthy weight            | Obesity increases cancer risk |
| Limit high-calorie foods/beverages | Prevention focus              |
| Exercise 150 min/week              | Cancer prevention             |
| Balanced diet                      | Supports health               |

## Cancer Treatment Nutrition

| Problem                 | Teaching                          |
| ----------------------- | --------------------------------- |
| Weight loss/poor intake | High-calorie, high-protein foods  |
| Poor appetite           | Small frequent meals              |
| Metallic taste          | Plastic utensils                  |
| Mucositis               | Soft, moist, cold/room-temp foods |
| Nausea                  | Small meals; avoid strong odors   |
| Infection               | Increased metabolic needs         |

## Mucositis / Stomatitis

Better choices:

* Cold/room-temperature foods
* Soft, bland, moist foods
* High-calorie shakes
* High-protein drinks
* Popsicles if nonacidic

Avoid:

* Spicy foods
* Salty foods
* Citrus
* Tomato juice/tomato-based foods
* Acidic foods
* Rough/crunchy foods
* Very hot foods/drinks
* Glycerin swabs

Oral care:

* Soft toothbrush
* Saline/baking soda rinses
* Increase fluids as tolerated

## Neutropenic Food Safety

| Teaching          | ATI Point                             |
| ----------------- | ------------------------------------- |
| Leftovers         | Use within 24 hr                      |
| Hot foods         | Keep above 140°F / 60°C               |
| Thawing           | Thaw in refrigerator                  |
| Raw foods         | Avoid unsafe/undercooked foods        |
| Home-canned foods | Cook appropriately; follow safe rules |

## Wound Healing

Needs:

* Protein
* Calories
* Vitamin C
* Zinc
* Iron
* Fluids

<h2 class="section-divider" id="pregnancy-peds-condensed">Pregnancy, Lactation, Infant, and Pediatric Nutrition</h2>

## Pregnancy Weight Gain

| Prepregnancy BMI | Category    | Recommended Gain |
| ---------------- | ----------- | ---------------- |
| Less than 18.5   | Underweight | 28–40 lb         |
| 18.5–24.9        | Normal      | 25–35 lb         |
| 25.0–29.9        | Overweight  | 15–25 lb         |
| 30.0 or greater  | Obese       | 11–20 lb         |

## Pregnancy Nutrition

| Nutrient/Issue  | ATI Point                                       |
| --------------- | ----------------------------------------------- |
| **Iron**        | 30 mg/day                                       |
| **Folate**      | Neural tube development                         |
| **Protein**     | Increased needs                                 |
| **Hyperemesis** | Dehydration, electrolyte imbalance, weight loss |
| **Fat intake**  | Reduce excess fat if needed                     |

## Breastfeeding / Breast Milk

| Topic               | Teaching                                           |
| ------------------- | -------------------------------------------------- |
| Frequency           | On demand, about 8–12 times/day                    |
| First day           | May nurse 5 min each breast                        |
| Early breastfeeding | Avoid unnecessary formula/water/artificial nipples |
| Thawed breast milk  | Use within 24 hr                                   |
| Refreezing          | Do not refreeze                                    |
| Warming             | Warm running water or refrigerator; no microwave   |
| Milk left in bottle | Discard                                            |

## Infant / Pediatric Nutrition

| Topic             | ATI Point                                                               |
| ----------------- | ----------------------------------------------------------------------- |
| Birth weight      | Regained by about 2 weeks                                               |
| 1-year growth     | Triples birth weight; length increases 50%                              |
| Solid foods       | Introduce one new food every 3–5 days                                   |
| Whole milk        | Not before 12 months; whole milk until age 2                            |
| Cleft lip feeding | One-way valve bottle, high-flow nipple, cheek support, frequent burping |

## Child / Adolescent Healthy Choices

| Instead Of             | Better Choice                 |
| ---------------------- | ----------------------------- |
| Refried beans          | Pinto beans                   |
| White sauce pasta      | Red sauce pasta               |
| Sausage                | Canadian bacon                |
| Sweetened yogurt       | Plain fat-free yogurt + fruit |
| Fatty ground beef      | 90% lean ground beef          |
| High-fat cheese        | ≤3 g fat/serving              |
| High-sat-fat margarine | ≤2 g saturated fat/Tbsp       |

<h2 class="section-divider" id="older-adult-condensed">Older Adult Nutrition</h2>

## Older Adult Priorities

| Issue               | ATI Point                          |
| ------------------- | ---------------------------------- |
| Lower calorie needs | Need nutrient-dense foods          |
| Protein needs       | Still important for muscle/healing |
| Dehydration         | Thirst response may be decreased   |
| Dementia            | Poor intake risk                   |
| Bone health         | Calcium + vitamin D                |
| Surgery/illness     | Increased calorie/protein needs    |

## Protein Needs

| Client                | Protein Need     |
| --------------------- | ---------------- |
| Healthy older adult   | 1–1.2 g/kg/day   |
| Chronic/acute illness | 1.2–1.5 g/kg/day |

## Older Adult Teaching

* Offer small, frequent nutrient-dense meals
* Monitor weight and intake
* Encourage fluids unless contraindicated
* Reduce distractions for dementia
* Offer finger foods for dementia
* Calcium carbonate with meals
* Vitamin D may be 1,000–2,000 IU/day if recommended
* Post-surgery intake may need at least 1,500 kcal/day

<h2 class="section-divider" id="culture-condensed">Cultural Considerations and Special Diets</h2>

## Best Nursing Approach

* Ask about food preferences and restrictions
* Avoid assumptions
* Respect religious/cultural practices
* Assess safety with fasting, illness, pregnancy, diabetes, kidney disease, and medications
* Collaborate with dietitian/provider when needed

## Common Religious / Cultural Diet Clues

| Group/Practice        | Common Nutrition Consideration                                   |
| --------------------- | ---------------------------------------------------------------- |
| Muslim / Islam        | Halal; avoid pork/alcohol; Ramadan fasting                       |
| Jewish / Judaism      | Kosher; avoid pork/shellfish; separate meat/dairy                |
| Hindu                 | Many avoid beef; some vegetarian                                 |
| Buddhist              | Some vegetarian/vegan; practices vary                            |
| Sikh                  | Some vegetarian; some avoid halal/kosher-slaughtered meat        |
| Seventh-day Adventist | Many vegetarian; may avoid pork/shellfish/alcohol/caffeine       |
| Catholic / Orthodox   | Fasting or meat/dairy restrictions during religious periods      |
| LDS / Mormon          | Avoid alcohol, coffee, tea                                       |
| Jain                  | Strict vegetarian; may avoid root vegetables                     |
| Rastafarian           | Ital diet; may avoid processed foods, pork, shellfish, additives |

## Vegetarian / Vegan Nutrition

| Nutrient Concern | Sources / Teaching                                       |
| ---------------- | -------------------------------------------------------- |
| Protein          | Beans, lentils, soy, nuts, seeds, dairy/eggs if included |
| B12              | Fortified foods or supplement for vegans                 |
| Iron             | Beans, fortified grains, greens; take with vitamin C     |
| Calcium          | Dairy, fortified plant milk, tofu, greens                |
| Vitamin D        | Fortified foods or supplement                            |
| Zinc             | Beans, nuts, seeds, whole grains                         |

<div class="danger">
Strict vegan diets require attention to <strong>vitamin B12</strong>.
</div>

<h2 class="section-divider" id="enteral-parenteral-condensed">Enteral and Parenteral Nutrition</h2>

## Enteral Nutrition

Enteral nutrition uses the GI tract through a feeding tube.

| Topic             | ATI Point                                            |
| ----------------- | ---------------------------------------------------- |
| Initial placement | Verify by radiograph/X-ray before first feeding      |
| HOB               | 30–45 degrees during and about 1 hr after feeding    |
| Aspiration        | Coughing, choking, crackles, dyspnea                 |
| Gastric residuals | Delayed emptying/aspiration risk                     |
| Diarrhea          | Rate may be too fast; decrease rate if prescribed    |
| Meds              | Do not crush enteric-coated or extended-release meds |
| Flushes           | 30–50 mL before/after meds to maintain patency       |

## Parenteral Nutrition / TPN

TPN bypasses the GI tract and provides nutrition through the bloodstream.

| Topic              | ATI Point                                                |
| ------------------ | -------------------------------------------------------- |
| Central line       | Infection and thrombosis risk                            |
| Hyperglycemia      | TPN contains dextrose; monitor glucose                   |
| Sudden stop        | Hypoglycemia risk                                        |
| Hypoglycemia signs | Confusion, cool clammy skin, shakiness, erratic behavior |
| Electrolytes       | Low magnesium risk/electrolyte shifts                    |
| Lipids             | May contain egg phospholipid; check egg allergy          |
| Discontinue TPN    | When PO intake is >60% calories if prescribed            |

<h2 class="section-divider" id="med-food-condensed">Medication and Food Interactions</h2>

## High-Yield Interactions

| Medication / Topic                 | Food or Nutrient Concern | ATI Teaching                                              |
| ---------------------------------- | ------------------------ | --------------------------------------------------------- |
| **Warfarin**                       | Vitamin K                | Keep intake consistent                                    |
| **ACE inhibitors**                 | Potassium                | Hyperkalemia risk; watch high-potassium foods             |
| **Captopril**                      | Potassium                | Cantaloupe is high potassium                              |
| **Salt substitutes**               | Potassium                | Avoid if CKD/hyperkalemia risk unless approved            |
| **MAOIs**                          | Tyramine                 | Avoid aged cheese/tyramine foods                          |
| **Iron**                           | Vitamin C                | Vitamin C improves absorption                             |
| **Iron**                           | Milk/calcium/tea/coffee  | Can decrease absorption                                   |
| **Liquid iron**                    | Teeth staining           | Use straw                                                 |
| **Calcium carbonate**              | Food                     | Take with meals                                           |
| **Levothyroxine**                  | Food/calcium/iron        | Take empty stomach; separate minerals                     |
| **Bisphosphonates**                | Food/esophagus           | Plain water only; stay upright                            |
| **Tetracyclines/fluoroquinolones** | Dairy/minerals           | Separate from dairy/calcium/iron/magnesium                |
| **Lithium**                        | Sodium/fluid             | Keep intake consistent; dehydration increases toxicity    |
| **Loop/thiazide diuretics**        | Potassium loss           | Monitor potassium                                         |
| **Potassium-sparing diuretics**    | Potassium retention      | Avoid excess potassium/salt substitutes                   |
| **Digoxin**                        | Low potassium            | Increases toxicity risk                                   |
| **Corticosteroids**                | Glucose/sodium/bone      | Monitor glucose; limit sodium; calcium/vit D if long-term |
| **Isoniazid**                      | Vitamin B6               | Pyridoxine may be prescribed                              |
| **Phenytoin**                      | Tube feeds/vitamins      | Tube feeds may reduce absorption                          |
| **Grapefruit**                     | Drug metabolism          | Avoid if label/provider says so                           |
| **Insulin**                        | Meal timing              | Skipping meals can cause hypoglycemia                     |
| **Metformin**                      | Contrast dye             | May need to be held around contrast studies               |
| **Lipid emulsions**                | Egg allergy              | Check allergy                                             |

<h2 class="section-divider" id="foods-to-know-condensed">Foods to Know</h2>

## Food Choice Clues

| If ATI Mentions...          | Choose / Avoid                                                                |
| --------------------------- | ----------------------------------------------------------------------------- |
| **Celiac disease**          | Choose rice, corn, quinoa, tapioca, potato flour; avoid wheat/barley/rye/malt |
| **Warfarin**                | Keep leafy greens consistent                                                  |
| **ACE inhibitor**           | Watch high potassium foods like cantaloupe                                    |
| **MAOI**                    | Avoid aged cheese/tyramine                                                    |
| **Iron**                    | Take with vitamin C; avoid milk/calcium at same time                          |
| **Liquid iron**             | Use straw                                                                     |
| **Mucositis**               | Cold, soft, bland foods                                                       |
| **Dumping syndrome**        | Avoid simple sugars; fluids between meals                                     |
| **Gastric bypass**          | Protein first                                                                 |
| **COPD weight loss**        | High-calorie, high-protein foods                                              |
| **Constipation**            | Fiber + fluids                                                                |
| **Diverticulosis**          | High fiber                                                                    |
| **Diverticulitis flare**    | Low fiber/low residue if prescribed                                           |
| **Ileostomy**               | Fluids, salt if prescribed, stool-thickening foods                            |
| **Neutropenia**             | Food safety; leftovers within 24 hr                                           |
| **Pregnancy**               | Iron, folate, protein                                                         |
| **Older adult bone health** | Calcium and vitamin D                                                         |
| **Vegan diet**              | Vitamin B12 fortified foods/supplement                                        |

## Stool-Thickening Foods

* Applesauce
* Bananas
* Pasta
* Rice
* Potatoes
* Peanut butter
* Bread
* Pectin-containing foods

## High-Fiber Foods

* Whole grains
* Bran
* Beans
* Lentils
* Fruits/vegetables with skins
* Dried peas
* Nuts/seeds if tolerated

## Low-Residue Foods

* White rice
* White bread
* Refined pasta
* Eggs
* Tender meat
* Strained juices
* Cooked vegetables without skins/seeds if allowed
* Canned fruit without skins if allowed

| Morning hyperglycemia + nighttime hypoglycemia       | Somogyi phenomenon                                         |

| Plain rice, corn, quinoa, potato flour               | Gluten-free                                                |
| Gastrectomy + diarrhea/dizziness after meals         | Dumping syndrome                                           |
| Simple sugars after gastric surgery                  | Worsen dumping syndrome                                    |
| Tube feeding + diarrhea                              | Rate may be too fast                                       |
| Tube feeding + coughing/crackles                     | Aspiration risk                                            |
| TPN stopped suddenly                                 | Hypoglycemia risk                                          |
| TPN + egg allergy                                    | Lipid emulsion concern                                     |
| Warfarin + leafy greens                              | Consistent vitamin K                                       |
| Captopril + cantaloupe                               | Hyperkalemia risk                                          |
| MAOI + cheese                                        | Tyramine/hypertensive crisis                               |
| Mouth sores                                          | Soft, cold/room-temp foods                                 |

| Pressure injury/wound                                | Protein and calories                                       |

## Common Foods and What They Have

| Food                                         | What It Has / Why It Matters                  |
