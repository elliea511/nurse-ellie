<h2 class="section-divider" id="nutrition-basics--foundations">Nutrition Basics & Foundations</h2>

Nutrition questions on ATI are usually not testing random food memorization. They are testing whether you understand how a nurse evaluates nutritional status, how food moves through the GI tract, how nutrients are broken down and absorbed, and what happens when intake, digestion, or absorption is impaired.

**Main nutrition flow:**  
**assess → eat → digest → absorb → use → recognize problems**

Almost every question in this section connects back to one part of that chain.

---

<h3 id="big-picture">Big Picture: What ATI Actually Tests</h3>

| Question Type | What ATI Is Testing |
| --- | --- |
| **Assessment** | Can you identify nutrition risk using history, anthropometrics, and labs? |
| **Food choice** | Can you pick the safest or best food for the client's condition? |
| **Deficiency** | Can you connect a symptom to a missing nutrient? |
| **Priority** | Can you recognize malnutrition, dehydration, fluid overload, or electrolyte imbalance? |
| **Teaching** | Can you give realistic instructions the client can actually follow? |
| **Medication-food interaction** | Can you identify nutrients that interact with medications? |
| **Digestion/absorption** | Can you identify the organ, enzyme, or nutrient involved? |

**ATI clue:** Do not just memorize the food. Ask: **What does this nutrient do? Where is it digested or absorbed? Why does this client need or avoid it?**

<h3 id="nutrition-assessment">Nutrition Assessment & Screening</h3>

Before teaching or planning a diet, the nurse has to determine whether a nutrition problem actually exists. ATI tests this using both subjective and objective data, plus formal nutrition screening tools.

#### Subjective vs. Objective Nutrition Data

| Data Type | What It Includes |
| --- | --- |
| **Subjective** | Diet history, appetite changes, chewing/swallowing difficulty, GI symptoms, food allergies/intolerances, cultural/religious practices, finances, living situation, medications, alcohol use |
| **Objective** | Anthropometrics, physical exam findings, and labs |

**Objective data examples:** height, weight, BMI, waist circumference, skin, hair, oral mucosa, muscle mass, edema, albumin, prealbumin, CBC, glucose, and electrolytes.

**Key idea:** No single measurement proves malnutrition. ATI wants the nurse to look at the **whole picture**: history + physical findings + labs.

#### BMI Categories

| BMI (kg/m²) | Category |
| --- | --- |
| Less than 18.5 | Underweight |
| 18.5–24.9 | Normal/healthy weight |
| 25.0–29.9 | Overweight |
| 30.0 or greater | Obese |

**BMI limitation:** BMI cannot tell muscle from fat mass, and it does not detect micronutrient deficiencies. A client can have a normal BMI and still be malnourished.

#### Significant Weight Change

| Time Frame | Weight Loss Considered Significant |
| --- | --- |
| **1 week** | Greater than 2% |
| **1 month** | Greater than 5% |
| **3 months** | Greater than 7.5% |
| **6 months** | Greater than 10% |

Unintentional weight loss at these levels should be reported. It is a red flag for malnutrition even if BMI still looks normal.

#### Anthropometric Measurements

| Measurement | What It Tells You |
| --- | --- |
| **Weight/height trends** | Best when trended over time using the same scale, similar clothing, and same time of day |
| **Waist circumference** | Central adiposity; cardiometabolic risk increases above about 40 in for men or 35 in for women |
| **Waist-to-hip ratio** | Central fat distribution and cardiovascular risk |
| **Mid-upper arm circumference (MUAC)** | Quick screen for muscle/fat wasting, useful when weight/height cannot be obtained |
| **Skinfold thickness** | Estimates subcutaneous fat stores, often triceps or subscapular sites |

#### Formal Screening Tools

| Tool | What It Does |
| --- | --- |
| **Malnutrition Screening Tool (MST)** | Quick bedside screen using recent weight loss and appetite/intake change |
| **Mini Nutritional Assessment (MNA)** | Used specifically for **older adults**; includes intake, weight loss, mobility, and BMI |
| **Subjective Global Assessment (SGA)** | Uses history and physical exam to classify nutrition status as well-nourished, moderately malnourished, or severely malnourished |

#### Physical Exam Clues by Deficiency

| Finding | Possible Deficiency |
| --- | --- |
| Pale conjunctiva | Iron, B12, or folate deficiency anemia |
| Bitot spots, dry conjunctiva/xerosis | Vitamin A deficiency |
| Swollen, bleeding gums | Vitamin C deficiency |
| Inflamed, smooth tongue/glossitis | B-vitamin deficiency, especially B12, folate, or niacin |
| Pale, brittle, spoon-shaped nails | Iron deficiency |
| Dry, brittle hair; easily plucked | Protein-calorie malnutrition |
| Poor wound healing | Protein, vitamin C, or zinc deficiency |
| Paresthesia, unsteady gait | B12 deficiency; can be irreversible if untreated |

<h3 id="gi-tract-flow">The GI Tract: Simple Flow</h3>

Food moves through the GI tract in order: **mouth → esophagus → stomach → small intestine → large intestine**.

| GI Area | Main Job | ATI Clue |
| --- | --- | --- |
| **Mouth** | Mechanical digestion starts with chewing; saliva starts carb digestion | **Amylase** begins breaking down starches |
| **Esophagus** | Moves food to stomach | Think swallowing safety, aspiration, dysphagia |
| **Stomach** | Churns food; acid/enzymes begin protein digestion | Protein digestion begins here |
| **Small intestine** | Main site of digestion and absorption | Most nutrients are absorbed here |
| **Pancreas** | Releases digestive enzymes into small intestine; also makes insulin | Amylase, lipase, proteases |
| **Liver / gallbladder** | Liver makes bile; gallbladder stores/releases it | Bile helps fat digestion but is not an enzyme |
| **Large intestine** | Absorbs water and forms stool | Think constipation, diarrhea, ostomy output |

**High-yield:** The **small intestine** is the main site of nutrient absorption. The **large intestine** mainly absorbs water and forms stool.

#### Follow the Food Through the GI Tract

Food enters the mouth, where chewing breaks it down and saliva begins carbohydrate digestion. It moves through the esophagus into the stomach, where churning and acid begin protein digestion. From there it enters the small intestine, which is the main site of digestion and absorption. Pancreatic enzymes and bile do most of the chemical work here. Leftover waste moves into the large intestine, where water is absorbed and stool forms.

**Pediatric note:** An infant's intestine is **proportionally longer relative to body length** than an adult's. This gives infants more absorptive surface area per pound of body weight to support rapid growth. It also means more surface area is exposed to fluid loss during vomiting or diarrhea, which helps explain why GI illness can dehydrate infants quickly.

<h3 id="digestion-vs-absorption">Digestion vs. Absorption</h3>

| Term | Meaning | Example |
| --- | --- | --- |
| **Digestion** | Breaking food into smaller parts | Protein breaks down into amino acids |
| **Absorption** | Moving nutrients from intestine into blood | Amino acids move into the bloodstream |

**ATI clue:** If the question says **damaged villi**, think **absorption problem**. If it says **low pancreatic enzymes**, think **digestion problem**.

#### Digestion vs. Absorption by Condition

| Condition | Digestion or Absorption Issue? | Why |
| --- | --- | --- |
| Pancreatitis / cystic fibrosis | Digestion | Pancreatic enzymes are impaired |
| Celiac disease / Crohn's disease | Absorption | Villi are damaged or inflamed |
| Bile duct obstruction | Fat digestion | Bile cannot reach the intestine |
| Lactose intolerance | Digestion | Lactase is low |
| Short bowel syndrome | Absorption | Reduced surface area for nutrient uptake |

<h3 id="macronutrients">Macronutrients</h3>

Macronutrients are nutrients needed in large amounts. They provide energy and include **carbohydrates, protein, and fat**.

| Macronutrient | % of Daily Calories | Main Job | Enzyme | Broken Down Into |
| --- | --- | --- | --- | --- |
| **Carbohydrates** | 45–65% | Preferred energy source | **Amylase** | Glucose/simple sugars |
| **Protein** | 10–35% | Tissue repair, immunity, muscle | **Proteases** | Amino acids |
| **Fat** | 20–35% | Concentrated energy, vitamin absorption | **Lipase** + bile | Fatty acids + glycerol |

**Memory:** **Amylase = carbs. Protease = protein. Lipase = fat.** Bile helps emulsify fat but is not an enzyme.

#### Caloric Value Per Gram

| Nutrient | Calories per Gram |
| --- | --- |
| **Carbohydrate** | 4 kcal/g |
| **Protein** | 4 kcal/g |
| **Fat** | 9 kcal/g |
| **Alcohol** | 7 kcal/g |

Fat has more than double the calories per gram of carbs or protein. This matters for calorie-control questions and label calculations.

**Note:** These percentages are general ranges for healthy adults. Kidney disease, burns, wounds, diabetes, pancreatitis, pregnancy, malnutrition, and other disease processes can change the plan.

---

<h4 id="carbohydrates">Carbohydrates</h4>

Carbohydrates are the body's preferred energy source. They break down into glucose, which fuels the brain, muscles, and cells. The general recommended minimum is about **130 g/day** to supply the brain with adequate glucose.

| Carbohydrate Class | Examples | Notes |
| --- | --- | --- |
| **Monosaccharides** | Glucose, fructose, galactose | Simplest form; absorbed directly |
| **Disaccharides** | Sucrose, lactose, maltose | Must be broken into monosaccharides before absorption |
| **Polysaccharides/starches** | Grains, legumes, root vegetables | Digest more slowly than simple sugars |
| **Fiber** | Whole grains, fruits, vegetables | Not digested/absorbed for energy |

#### Simple vs. Complex Carbohydrates

| Type | Digestion Speed | ATI Relevance |
| --- | --- | --- |
| **Simple carbs** | Digest and absorb quickly | Raise glucose quickly; major dumping syndrome trigger |
| **Complex carbs** | Digest more slowly | Usually the better routine choice; typically more fiber and nutrients |

**Protein-sparing effect:** When carbohydrate/calorie intake is adequate, the body can use protein for healing, immunity, and muscle maintenance instead of burning it for energy. Too few carbs can cause protein breakdown for fuel, which worsens wound healing and muscle wasting.

**Teaching point:** ATI usually does not want a client to eliminate all carbs. Safer teaching is **consistent carbohydrate intake** and choosing more complex carbs.

<h4 id="lactose-intolerance">Lactose Intolerance</h4>

Lactose is the disaccharide found in milk and dairy. It is normally broken down by **lactase**, which is made at the brush border of the small intestine. Lactase breaks lactose into glucose and galactose. When lactase is low, undigested lactose stays in the bowel, draws water in, and gets fermented by colonic bacteria.

| Concept | Detail |
| --- | --- |
| **Cause** | Insufficient lactase enzyme; can be genetic/age-related, or temporary after GI illness or bowel surgery |
| **Symptoms** | Bloating, gas, abdominal cramping, diarrhea, usually within a couple hours after dairy |
| **Not the same as milk allergy** | Lactose intolerance is an enzyme/digestion problem; milk allergy is an immune reaction to milk protein and can be life-threatening |

**ATI clue:** Gas, bloating, cramping, or diarrhea after dairy with no hives, swelling, or respiratory symptoms = **lactase deficiency**, not allergy.

#### Lactose Intolerance Teaching

| Teaching | Why It Helps |
| --- | --- |
| Use lactose-free or lactose-reduced dairy | Reduces symptoms while preserving nutrition |
| Take lactase enzyme supplement before dairy, if used | Replaces missing enzyme |
| Try hard, aged cheeses and yogurt | Often better tolerated than milk because fermentation/aging reduces lactose |
| Spread small dairy servings across the day | May be better tolerated than one large serving |
| Maintain calcium and vitamin D intake | Use fortified plant milks, leafy greens, calcium-set tofu, or supplements if dairy is reduced |

The goal is not just “avoid milk.” The goal is to manage symptoms while still getting enough **calcium and vitamin D**.

<h4 id="protein">Protein</h4>

Protein is needed for tissue repair, wound healing, immune function, muscle maintenance, nitrogen balance, and fluid balance through oncotic pressure. Protein also supports **transport**: hemoglobin carries oxygen, and albumin carries many medications and nutrients through the blood.

Protein can be used as a backup energy source when carbohydrate and fat stores run out, but this is not ideal.

#### Amino Acids

There are 20 amino acids that make up protein:

| Type | Meaning |
| --- | --- |
| **Nonessential amino acids** | The body can manufacture 11 of them |
| **Essential amino acids** | The body must get 9 from food |

**Nine essential amino acids:** histidine, isoleucine, leucine, lysine, methionine, phenylalanine, threonine, tryptophan, and valine.

When protein is broken down for energy, amino acids are further broken down into **ammonia**, a nitrogen waste product that the liver and kidneys must clear. This is why protein is not the body’s preferred energy source.

#### Protein Types

| Protein Type | Meaning |
| --- | --- |
| **Complete** | Contains all 9 essential amino acids |
| **Incomplete** | Missing one or more essential amino acids |
| **Complementary proteins** | Two incomplete proteins eaten together that supply all essential amino acids |

**ATI clue:** If **cottage cheese** or **soy** is an answer option for “best protein source,” it is often correct because both are complete proteins.

#### Protein Needs by Client Situation

| Increased Protein Needs | Restricted/Decreased Protein Needs |
| --- | --- |
| Wound healing, pressure injuries, burns, surgery, infection, cancer, pregnancy/lactation, COPD, older adulthood, malnutrition, Parkinson’s disease, dialysis | Some chronic kidney disease clients not yet on dialysis; hepatic encephalopathy in advanced liver disease |

**High-yield:** Low protein → low oncotic pressure → fluid leaks into tissue. Poor intake + weight loss + lower-extremity edema = think protein deficiency, even if the client does not look underweight.

<h4 id="fat">Fat</h4>

Fat provides concentrated energy, insulation, organ protection, and helps absorb vitamins **A, D, E, and K**.

| Fat Type | Effect | ATI Relevance |
| --- | --- | --- |
| **Saturated fat** | Raises LDL cholesterol | Limit intake |
| **Trans fat** | Raises LDL and lowers HDL | Avoid as much as possible |
| **Unsaturated fat** | Generally heart-protective | Preferred fat type in appropriate portions |

**ATI clue:** Greasy, floating, foul-smelling stools = **fat malabsorption**. Think impaired lipase, pancreatitis, cystic fibrosis, impaired bile flow, gallstones, bile duct obstruction, or liver disease.

<h3 id="enzyme-quick-map">Enzyme & Symptom Quick Map</h3>

| If ATI Mentions... | Think... |
| --- | --- |
| Carbs / starches | **Amylase** |
| Protein breakdown | **Protease** |
| Fat / lipids | **Lipase** |
| Fat emulsification | **Bile**; not an enzyme |
| Lactose / dairy sugar | **Lactase** |
| Greasy, floating stools | Fat malabsorption |
| Gas/diarrhea after dairy | Lactase deficiency |
| Clay-colored stool + jaundice | Bile flow problem |
| Pancreatitis / cystic fibrosis | Pancreatic enzyme impairment affecting carbs, fat, and protein |
| Celiac disease | Damaged villi → malabsorption |

#### Pancreatic Enzyme Replacement

Used for pancreatic insufficiency, chronic pancreatitis, cystic fibrosis, or after pancreatic surgery.

| Teaching | Why |
| --- | --- |
| Take enzymes **with meals and snacks** | Enzymes only work if present when food is eaten |
| Do not crush or chew enteric-coated capsules | Protects medication design and prevents irritation |
| Monitor stool pattern and weight | Continued greasy stools or weight loss can mean poor digestion/absorption |

<h3 id="fiber">Fiber</h3>

Fiber is a carbohydrate the body cannot fully digest. It does not provide energy the same way glucose does, but it affects GI function, cholesterol, and glucose control.

| Fiber Type | Function | Common Sources |
| --- | --- | --- |
| **Soluble fiber** | Dissolves in water, forms a gel, lowers LDL, slows glucose absorption | Oats, beans, apples, citrus, barley |
| **Insoluble fiber** | Adds bulk and speeds movement through the colon | Whole wheat, bran, vegetables, skins |

**Memory:** **Soluble = gel, slows things down. Insoluble = bulk, speeds things up.**

General adult recommendation is about **25–30 g/day**. Increase fiber gradually and pair it with more fluid. Fiber without enough water can worsen constipation and bloating.

---

<h3 id="water">Water as a Nutrient</h3>

Water is a nutrient because it is required for digestion, absorption, temperature regulation, waste removal, and metabolic reactions.

| General Guideline | Important Exceptions |
| --- | --- |
| About **25–35 mL/kg/day**, or roughly **2–3 L/day** for many healthy adults | Needs increase with fever, vomiting, diarrhea, wound drainage, and hot environments; fluids may be restricted in heart failure, CKD, or SIADH |

ATI rarely asks for an exact water number. It usually tests whether you can recognize **dehydration vs. fluid overload** and whether fluids should be encouraged or restricted for a diagnosis.

<h3 id="rda-dri">RDA, DRI, and Nutrition Recommendations</h3>

| Term | Meaning |
| --- | --- |
| **DRI** | Dietary Reference Intake; umbrella term that includes EAR, RDA, AI, and UL |
| **EAR** | Estimated Average Requirement; intake meeting needs of 50% of a healthy group; basis used to calculate RDA |
| **RDA** | Recommended Dietary Allowance; amount that meets the needs of most healthy people in a group |
| **AI** | Adequate Intake; used when there is not enough evidence for an RDA |
| **UL** | Tolerable Upper Intake Level; highest amount unlikely to cause harm |
| **AMDR** | Acceptable Macronutrient Distribution Range; carb/protein/fat percentage ranges |

RDA values are for generally healthy people. Pregnancy, lactation, illness, wound healing, and malnutrition can increase needs above standard RDA values.

<h3 id="vitamins">Micronutrients: Vitamins</h3>

Vitamins are micronutrients needed in smaller amounts. ATI usually tests their function, deficiency symptoms, toxicity risk, and food/medication interactions.

#### Fat-Soluble vs. Water-Soluble Vitamins

| Type | Vitamins | Main Point |
| --- | --- | --- |
| **Fat-soluble** | A, D, E, K | Stored in the body; higher toxicity risk; need fat for absorption |
| **Water-soluble** | B-complex, C | Not stored in large amounts; excess usually excreted in urine |

**Memory:** Fat-soluble = **A, D, E, K**. Fat malabsorption increases risk for deficiency of all four.

#### High-Yield Vitamin Functions

| Vitamin | Main ATI Function | High-Yield Clue |
| --- | --- | --- |
| **Vitamin A** | Night vision, skin/mucous membrane integrity, immune support | Night vision/dim light questions; Bitot spots/dry conjunctiva |
| **Lutein** | Eye health; slows progression of age-related macular degeneration | Carotenoid related to vitamin A |
| **Thiamine (B1)** | Carbohydrate metabolism and nerve function | Deficiency linked to alcohol use disorder and Wernicke-Korsakoff |
| **Vitamin B6** | Heme formation and neurotransmitter synthesis | Isoniazid can deplete B6, so they are often given together |
| **Folate (B9)** | RBC formation and fetal neural tube development | Critical before conception and in early pregnancy; deficiency causes megaloblastic anemia |
| **Vitamin B12** | RBC formation, nervous system/myelin maintenance, DNA synthesis | Risk with strict vegan diet, older adults/atrophic gastritis, bariatric surgery; deficiency can cause irreversible neurologic damage |
| **Vitamin C** | Wound healing, collagen, immune function, improves iron absorption | Pair with iron; avoid if mucositis; deficiency causes bleeding gums and poor healing |
| **Vitamin D** | Calcium absorption and bone health | Older adults often need supplementation, commonly 1,000–2,000 IU/day |
| **Vitamin E** | Antioxidant; protects cell membranes | Rarely tested alone; remember it is fat-soluble |
| **Vitamin K** | Blood clotting | Warfarin clients need consistent vitamin K intake, not elimination |

**Warfarin teaching:** Do **not** teach the client to avoid all vitamin K. Teach **consistent intake** of vitamin K-containing foods week to week.

#### Vitamin B12 vs. Folate

Both B12 and folate deficiencies can cause megaloblastic anemia, but **only B12 deficiency causes neurologic symptoms** such as numbness, tingling, paresthesia, and gait changes.

Giving folic acid to a client with undiagnosed B12 deficiency can correct the anemia while neurologic damage continues to progress.

#### Classic Vitamin Deficiency Diseases

| Vitamin/Nutrient | Deficiency Disease | Classic Presentation |
| --- | --- | --- |
| **Vitamin A** | Night blindness | Poor adaptation to dim light; can progress to immune problems |
| **Vitamin B1/thiamine** | Beriberi / Wernicke-Korsakoff | Nerve damage, confusion; linked to alcohol use disorder |
| **Vitamin B3/niacin** | Pellagra | Scaly rash on sun-exposed skin, confusion, diarrhea |
| **Vitamin C** | Scurvy | Delayed wound healing, capillary fragility, bleeding/swollen gums |
| **Vitamin D** | Rickets in children / osteomalacia in adults | Bowed legs, fractures, malformed teeth, bone pain |
| **Vitamin B12** | Pernicious anemia | Megaloblastic anemia plus neurologic symptoms |
| **Iron** | Iron-deficiency anemia | Fatigue, pallor, spoon-shaped nails/koilonychia |

**Megadose caution:** High-dose vitamin C supplementation, around 1 g/day or more, can increase the risk of oxalate kidney stone formation. More is not always better.

<h3 id="minerals">Key Minerals</h3>

| Mineral | Main Function | High-Yield Clue |
| --- | --- | --- |
| **Calcium** | Bones, teeth, nerve transmission, muscle contraction | Calcium carbonate should be taken **with meals** to improve absorption |
| **Magnesium** | Nerve, muscle, and cardiac function | Clients on TPN are at risk for **low magnesium** |
| **Potassium** | Cardiac rhythm, muscle contraction, fluid balance | ACE inhibitors like captopril can cause hyperkalemia |
| **Sodium** | Primary regulator of extracellular fluid volume; nerve/muscle function and BP | Excess sodium worsens HTN and fluid retention; about 1,500 mg/day is a common BP-reduction target |
| **Phosphorus** | Bone/teeth formation, hormone regulation, energy metabolism/ATP | Not connected to night vision; restricted in advanced CKD |
| **Iron** | Hemoglobin formation and oxygen transport | Pregnancy: **30 mg/day**; pair with vitamin C; use a straw for liquid iron; expect dark stools |
| **Zinc** | Wound healing, immune function, taste/smell, growth | Deficiency causes poor healing, hair loss, altered taste, frequent infections |

#### Calcium Notes

General adult RDA for calcium is **1,000 mg/day** for ages 19–50, including most lactating clients. During lactation, calcium absorption from the gut increases to help meet demand without raising the RDA itself. Needs increase for postmenopausal women and older adults.

**Fortified milk:** ATI notes that fortified milk enhances calcium absorption. Milk supplies calcium, and added vitamin D helps the gut absorb it.

#### Sodium Notes

ATI frames sodium’s main job as controlling **extracellular fluid volume**. Water follows sodium. Excess sodium can worsen fluid retention in hypertension, heart failure, CKD, and cirrhosis with ascites.

<h3 id="malnutrition">Malnutrition: Recognizing It</h3>

ATI tests malnutrition through assessment cues, not just body weight.

#### Nutrition Labs

| Lab | Normal Range | Half-Life | ATI Focus |
| --- | --- | --- | --- |
| **Albumin** | 3.5–5.0 g/dL | About 18–20 days | Longer-term protein status; affected by inflammation, liver/kidney disease, and fluid status |
| **Prealbumin** | 15–36 mg/dL | About 2 days | Best marker for **severe or recent** malnutrition because it changes faster |

A client can have normal albumin but low prealbumin if malnutrition is recent. Albumin has not had enough time to fall yet.

#### Physical Findings Suggesting Malnutrition

| Physical Finding | What It Can Suggest |
| --- | --- |
| Lower extremity edema | Protein deficiency |
| Dry skin / dry conjunctiva | Poor nutrition or vitamin deficiency |
| Weak hand grasp | Muscle wasting or poor protein intake |
| Poor wound healing | Inadequate protein, vitamin C, or zinc |
| Unintentional weight loss meeting significant thresholds | Inadequate intake, illness, or malabsorption |

**Do not miss:** Edema in a malnourished client can be the strongest clue for protein deficiency, even if the client does not look underweight.

#### Marasmus vs. Kwashiorkor

| Type | Cause | Classic Presentation |
| --- | --- | --- |
| **Marasmus** | Chronic deficiency of calories and protein | Severe muscle/fat wasting, low weight, minimal edema |
| **Kwashiorkor** | Adequate calories but inadequate protein | Edema, especially abdominal/ascites; normal or near-normal weight can mask deficiency; hair/skin changes |

**High-yield difference:** Marasmus looks wasted. Kwashiorkor may not look underweight because edema can hide the protein deficiency.

<h3 id="braden-scale">Braden Scale: Nutrition Scoring</h3>

The Braden Scale assesses pressure injury risk. Nutrition is one of its six categories.

| Score | Category | ATI Clues |
| --- | --- | --- |
| **1** | Very Poor | Rarely eats a complete meal; only bites; poor fluid intake; may be NPO/clear liquids without nutrition support |
| **2** | Probably Inadequate | Rarely eats a complete meal; usually eats about half of food offered; limited protein; occasional supplements |
| **3** | Adequate | Eats more than half of most meals; occasionally refuses a meal; usually gets enough protein servings |
| **4** | Excellent | Eats most/all meals; rarely refuses; regular protein; no supplementation needed |

Score based on the client’s **usual intake pattern**, not one meal. A client who eats more than half of most meals, occasionally refuses one meal, and gets 4 protein servings/day scores **3 — Adequate**.

<h3 id="fluid-balance">Fluid Balance</h3>

Vomiting and diarrhea place the client at risk for **both fluid volume deficit and electrolyte imbalance**.

#### Fluid Volume Deficit vs. Excess

| Assessment Area | Fluid Volume Deficit | Fluid Volume Excess |
| --- | --- | --- |
| Weight | Loss | Rapid gain |
| Mucous membranes | Dry | Usually moist |
| Skin | Poor turgor | Edema |
| Heart rate | Tachycardia | May be increased |
| Blood pressure | Low / orthostatic | High |
| Lungs | Usually clear | Crackles possible |
| Urine | Dark, concentrated, decreased | Variable |

#### Why Infants and Older Adults Are at Greater Risk for Dehydration

ATI tests this as a lifespan physiology concept. Both groups are high risk, but for different reasons.

| Factor | Infants | Older Adults |
| --- | --- | --- |
| **Total body water** | Higher percentage of body weight is water, about 70–80%, but more is in easily-lost extracellular space | Total body water declines with age as muscle decreases and fat increases |
| **Kidney function** | Immature kidneys; limited ability to concentrate urine | Decreased concentrating ability; less able to conserve water |
| **Intestinal length** | Intestine is proportionally longer relative to body length; more absorptive surface area but more surface exposed to fluid loss with vomiting/diarrhea | Not a major factor |
| **Thirst mechanism** | Cannot recognize or communicate thirst; dependent on caregiver | Blunted thirst response |
| **Surface area/metabolism** | Larger body-surface-area-to-mass ratio and higher metabolic rate cause greater insensible loss | Illness/fever has a larger impact because fluid reserve is smaller |
| **Fluid reserve** | Small total fluid volume, so losses become serious quickly | Smaller reserve; diuretics, mobility limits, and fear of incontinence can reduce intake |

**Exam version:** Infants lose fluid fast because of immature kidneys, high surface area, proportionally longer GI tract, and inability to ask for fluids. Older adults dehydrate because they have lower body water, reduced renal concentrating ability, and a blunted thirst response.

#### Dehydration Assessment by Age

| Group | Watch For |
| --- | --- |
| **Infants** | Sunken fontanelle, no tears when crying, fewer wet diapers, sunken eyes, lethargy, dry mucous membranes |
| **Older adults** | Confusion, weakness, dizziness, or falls; they may not complain of thirst |

<h3 id="weight-loss">Healthy Weight & Eating Patterns</h3>

Safe, realistic weight loss is usually **0.5 to 1 lb per week**. ATI favors sustainable behavior change over extreme restriction.

#### Weight Change Math

| Rule | Meaning |
| --- | --- |
| **3,500 kcal ≈ 1 lb body weight** | Used for ATI calorie-math questions |
| **500 kcal/day deficit** | About 1 lb lost per week |
| **1,000 kcal/day deficit** | About 2 lb lost per week |
| **500 kcal/day surplus** | About 1 lb gained per week |

**Formula:**  
`Total calorie change ÷ 3,500 = pounds gained or lost`

| Example | Math | Result |
| --- | --- | --- |
| Client cuts 300 kcal/day for 2 weeks | 300 × 14 = 4,200; 4,200 ÷ 3,500 | About 1.2 lb lost |
| Client eats 700 kcal/day above needs for 1 week | 700 × 7 = 4,900; 4,900 ÷ 3,500 | About 1.4 lb gained |

ATI uses the simple **3,500 kcal = 1 lb** rule. Do not overthink it with real-world metabolic changes.

#### Healthy Eating Teaching

| Teaching | Why It Works |
| --- | --- |
| Use a shopping list | Supports planning and prevents impulse buying |
| Eat low-calorie foods first | Helps fullness and portion control |
| Eat regular meals | Prevents extreme hunger and overeating |
| Control portions instead of eliminating foods | More sustainable long-term |

#### MyPlate, DASH, and Moderate Alcohol

| Topic | ATI Teaching |
| --- | --- |
| **MyPlate** | About half the plate fruits/vegetables, one quarter lean protein, one quarter grains, plus dairy/dairy alternative |
| **General fruit/vegetable target** | About 2.5 cups vegetables/day and 2 cups fruit/day; choose a variety of colors |
| **DASH diet** | Used for hypertension; emphasizes vegetables, fruit, low-fat/nonfat dairy, lean protein, whole grains, nuts/seeds/legumes, and reduced sodium/saturated fat/added sugar |
| **Moderate alcohol** | Up to 1 drink/day for women and up to 2 drinks/day for men, if the client chooses to drink |

#### DASH Servings Often Tested

| Food Group | DASH Amount |
| --- | --- |
| Vegetables | 4–6 servings/day |
| Fruit | 2–3 servings/day |
| Low-fat or nonfat dairy | 4–6 servings/day |
| Lean meat, fish, or poultry | 1.5–2.5 servings/day |
| Nuts, seeds, legumes | 3–6 servings/week |

**One drink equals:** 12 oz beer, 5 oz wine, or 1.5 oz distilled spirits over 80 proof. Alcohol teaching is always client- and condition-specific because it can affect nutrition and medication safety.

<h2 class="section-divider small" id="section-summary">Section Summary & Test-Taking Tips</h2>

| If the Question Says... | Think... |
| --- | --- |
| Most absorption happens where? | Small intestine |
| Starches / carbohydrates | Amylase |
| Protein breakdown | Proteases |
| Fat digestion | Lipase |
| Fat emulsification | Bile, not an enzyme |
| Dairy intolerance | Lactase |
| Gas/bloating/cramping/diarrhea after dairy, no hives or swelling | Lactase deficiency, not milk allergy |
| “Best” protein source | Complete protein, like cottage cheese or soy |
| Poor wound healing | Protein + vitamin C + zinc |
| Lower extremity edema + poor intake | Protein deficiency; consider kwashiorkor pattern |
| Night vision | Vitamin A; deficiency = night blindness |
| Bleeding gums, poor wound healing, capillary fragility | Vitamin C deficiency/scurvy |
| Scaly rash + confusion + diarrhea | Niacin/B3 deficiency/pellagra |
| Bowed legs or fractures in a child | Vitamin D deficiency/rickets |
| Anemia + neurologic symptoms | Vitamin B12 deficiency, not folate |
| Calories per gram | Carb = 4, protein = 4, fat = 9, alcohol = 7 |
| How many amino acids are essential? | 9; must come from diet |
| Warfarin | Consistent vitamin K intake |
| ACE inhibitors | Watch for high-potassium foods |
| Sodium’s main function | Controls extracellular fluid volume; water follows sodium |
| TPN | Monitor glucose and electrolytes, especially magnesium |
| Vomiting or diarrhea | Fluid volume deficit + electrolyte imbalance |
| Infant with poor intake/vomiting/diarrhea | High dehydration risk due to immature kidneys, inability to communicate thirst, high surface area |
| Older adult with confusion, weakness, or fall | Consider dehydration due to blunted thirst, lower body water, decreased renal concentrating ability |
| Severe/recent malnutrition | Prealbumin 15–36 mg/dL; half-life about 2 days |
| Albumin normal range | 3.5–5.0 g/dL; half-life about 18–20 days |
| Unintentional weight loss >5% in a month | Significant; investigate |
| 1 lb of weight gain/loss | 3,500 kcal surplus/deficit |
| Safe weight-loss goal | 0.5–1 lb/week |
